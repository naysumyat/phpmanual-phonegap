msql_field_table

(PHP 4, PHP 5)

msql_field_table — Obtém o nome da tabela pelo campo

Descrição

int msql_field_table ( resource $result , int $field_offset )

Retorna o nome da tabela pelo especificado campo.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

field_offset

O índice numérico. O field_offset começa em 1.

Valor Retornado

O nome da tabela em sucesso, ou FALSE em falha.
