readlink

(PHP 4, PHP 5)

readlink — Retornar o alvo de um link simbólico

Descrição

string readlink ( string $path )

readlink() faz o mesmo que a função do C.

Parâmetros

path

O caminho para o link simbólico.

Valor Retornado

Retorna o conteúdo do caminho do link simbólico ou FALSE em erro.

Exemplos

Exemplo #1 Exemplo de readlink()

<?php

// Exibe algo como /boot/vmlinux-2.4.20-xfs
echo readlink('/vmlinuz');

?>

Notas

Nota: esta função não é implementada na plataforma Windows
