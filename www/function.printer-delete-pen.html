printer_delete_pen

(PECL printer SVN)

printer_delete_pen — Exclui a caneta

Descrição

bool printer_delete_pen ( resource $handle )

Esta função exclui a caneta selecionada. Para um exemplo veja printer_select_pen(). Retorna TRUE em caso de sucesso ou FALSE em caso de falha. handle deve ser um manipilador válido para a caneta.
