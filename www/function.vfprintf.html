vfprintf

(PHP 5)

vfprintf — Escreve uma string formatada para um stream

Descrição

int vfprintf ( resource $handle , string $format , array $args )

Escreve uma string produzida de acordo com format para o resource do stream especificado por handle.

Funciona como fprintf() mas aceita um array de argumentos, ao invés de um número váriavel de argumentos.

Parâmetros

handle

format

Veja sprintf() para uma descrição do format.

args

Valor Retornado

Retorna o tamanho da string mostrada.

Exemplos

Exemplo #1 vfprintf(): preenchendo com zero os inteiros

<?php
if (!($fp = fopen('date.txt', 'w')))
    return;

vfprintf($fp, "%04d-%02d-%02d", array($year, $month, $day));
// will write the formatted ISO date to date.txt
?>
