imagerotate

(PHP 4 >= 4.3.0, PHP 5)

imagerotate — Gira uma imagem com o ângulo dado

Descrição

resource imagerotate ( resource $src_im , float $angle , int $bgd_color )

Gira a imagem src_im usando o ângulo angle dado em graus. bgd_color especifica a cor da zona descoberta após a rotação.

Nota: Esta função somente está disponível se o PHP estiver compilado com a versão embarcada da biblioteca GD.
