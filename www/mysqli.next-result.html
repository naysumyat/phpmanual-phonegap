mysqli::next_result

mysqli_next_result

(PHP 5)

mysqli::next_result -- mysqli_next_result — Prepara o próximo resultado de multi_query

Descrição

bool mysqli::next_result ( void )
bool mysqli_next_result ( mysqli $link )

Prepara o próximo conjunto de resultado de uma chamada anterior para mysqli_multi_query() que pode ser recuperado por mysqli_store_result() ou mysqli_use_result().

Parâmetros

link

Procedural style only: A link identifier returned by mysqli_connect() or mysqli_init()

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Veja mysqli_multi_query().

Veja Também

  • mysqli_multi_query() - Performs a query on the database
  • mysqli_more_results() - Verifica se há mais algum resultado de uma multi query
  • mysqli_store_result() - Transfers a result set from the last query
  • mysqli_use_result() - Initiate a result set retrieval
