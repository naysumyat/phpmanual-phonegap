log

(PHP 4, PHP 5)

log — Logaritmo natural

Descrição

float log ( float $arg [, float $base ] )

Se o parâmetro opcional base for informado, log() retorna logbase arg, senão log() retorna o logaritmo natural de arg.

Parâmetros

arg

O valor para calcular o logaritmo

base

O logaritmo base opcional para usar (padrão é 'e' e assim o logaritmo natural).

Valor Retornado

O logarimo de arg na base, se fornecido, ou o logaritmo natural.

Changelog

Versão Descrição
Since 4.3.0 O parâmetro opcional base tornou-se disponível. Para antigas versões você pode calcular o logaritmo na base b de um número n pela utilização da identidade matemática: logb(n) = log(n)/log(b), onde log é o logaritmo neperiano (ou natural).
