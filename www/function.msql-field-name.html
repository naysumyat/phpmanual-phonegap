msql_field_name

(PHP 4, PHP 5)

msql_field_name — Obtém o nome de um especificado campo de um resultado

Descrição

string msql_field_name ( resource $result , int $field_offset )

msql_field_name() obtém o nome do especificado campo.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

field_offset

O índice numérico. O field_offset começa em 1.

Valor Retornado

O nome do campo ou FALSE em falha.
