fam_suspend_monitor

(PHP 5 <= 5.0.5)

fam_suspend_monitor — Suspende monitoramento temporariamente

Descrição

bool fam_suspend_monitor ( resource $fam , resource $fam_monitor )

fam_suspend_monitor() temporariamente suspende o monitoramento de um recurso.

O monitoramento pode continuar, usando fam_resume_monitor() sem a necessidade de requisição completa de de um novo monitor.

Parâmetros

fam

Um resource representando a conexão para um serviço FAM retornado pela fam_open()

fam_monitor

Um resource retornado por uma das funções fam_monitor_XXX

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Veja Também

  • fam_cancel_monitor() - Finalizando monitoramento
  • fam_resume_monitor() - Recomeça um monitoramento suspendido
