is_infinite

(PHP 4 >= 4.2.0, PHP 5)

is_infinite —

Descrição

bool is_infinite ( float $val )

Retorna TRUE se val é infinito (positivo ou negativo), como por exemplo o resultado de log(0) ou qualquer outro valor muito grande para caber num float desta plataforma.

Parâmetros

val

O valor para verificar

Valor Retornado

TRUE se val é infinito, caso contrário FALSE.

  • is_nan() - Verifica se um valor não é um número
