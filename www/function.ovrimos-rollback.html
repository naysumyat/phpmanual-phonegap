ovrimos_rollback

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_rollback — Desfaz a transação

Descrição

bool ovrimos_rollback ( int $connection_id )

Desfaz a transação.

Parâmetros

connection_id

O identificador de conexão do Ovrimos, retornado pela ovrimos_connect().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
