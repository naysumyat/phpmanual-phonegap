XMLWriter::startComment

xmlwriter_start_comment

(PHP 5 >= 5.1.2, PECL xmlwriter >= 1.0.0)

XMLWriter::startComment -- xmlwriter_start_comment — Create start comment

Descrição

Estilo orientado à objeto

bool XMLWriter::startComment ( void )

Estilo procedural

bool xmlwriter_start_comment ( resource $xmlwriter )

Starts a comment.

Parâmetros

xmlwriter

Apenas para chamadas de procedimentos. O recurso XMLWriter resource que esta sendo modificado. Este recurso vem de uma chamada a xmlwriter_open_uri() ou xmlwriter_open_memory().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
