odbc_columns

(PHP 4, PHP 5)

odbc_columns — Lista os nomes das colunas na tabela especificada

Descrição

resource odbc_columns ( resource $connection_id [, string $qualifier [, string $schema [, string $table_name [, string $column_name ]]]] )

Lista todas as colunas no intervalo requerido.

Parâmetros

connection_id

O identificador de conexão.

qualifier

O quantificador.

schema

The owner.

table_name

O nome da tabela.

column_name

O nome da coluna.

O schema, table_name, e column_name aceitam padrões de busca ('%' para casar zero ou mais caracteres e '_' para casar um único caractere).

Valor Retornado

Retorna um identificador de resultado ODBC ou FALSE em erro.

O conjunto de resultados tem as seguintes colunas:

  • TABLE_QUALIFIER
  • TABLE_SCHEM
  • TABLE_NAME
  • COLUMN_NAME
  • DATA_TYPE
  • TYPE_NAME
  • PRECISION
  • LENGTH
  • SCALE
  • RADIX
  • NULLABLE
  • REMARKS

O conjunto de resultados é ordenado por TABLE_QUALIFIER, TABLE_SCHEM e TABLE_NAME.
