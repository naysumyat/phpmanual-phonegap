imagecolortransparent

(PHP 4, PHP 5)

imagecolortransparent — Define uma cor como tansparente

Descrição

int imagecolortransparent ( resource $image [, int $color ] )

Define a cor transparente na dada image.

Parâmetros

image

An image resource, returned by one of the image creation functions, such as imagecreatetruecolor().

color

Um identificador de cor criado com imagecolorallocate()

Valor Retornado

O identificador da nova (ou atual, se nenhuma for especificada) cor transparente é retornado.

Notas

Nota:

A transparência é copiada apenas com imagecopymerged() e imagens true color, não com imagecopy() ou imagens de paleta.

Nota:

A cor transparente é uma propriedade da imagem, a transparencia não é uma propriedade da cor. Uma vez que você definiu uma cor para ser a cor de transparência, todas as regiões da imagem que eram desenhadas com esta cor anteriormente serão transparentes.
