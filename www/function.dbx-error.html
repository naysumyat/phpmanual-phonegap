dbx_error

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5, PECL dbx >= 1.1.0)

dbx_error — Mostra a mensagem de erro da última função chamada no módulo

Descrição

string dbx_error ( objeto $link_identifier )

A função dbx_error() retorna a última mensagem de erro.

Parâmetros

link_identifier

O link do objeto DBX retornado por dbx_connect()

Valor Retornado

Retorna uma string contendo a mensagem de erro da última função chamda no módulo (ex: módulo mysql). Se houver múltiplas conexões no mesmo módulo, apenas o último erro é retornado para o módulo especificado no parâmetro link_identifier.

Exemplos

Exemplo #1 Exemplo da dbx_error()

<?php
$link   = dbx_connect(DBX_MYSQL, "localhost", "bd", "nome de usuário", "senha")
    or die ("Não foi possível conectar");

$resultado = dbx_query($link, "select id from tabela_inexistente");
if ( $resultado == 0 ) {
    echo dbx_error ($link);
}
dbx_close ($link);
?>

Notas

Nota:

Sempre refira-se à documentação do módulo-específico quando necessário.

A mensagem de erro para o Microsoft SQL Server é atualmente o resultado da função mssql_get_last_message().

A mensagem de erro para o Oracle (oci8) não foi implementado ainda.
