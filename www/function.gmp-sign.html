gmp_sign

(PHP 4 >= 4.0.4, PHP 5)

gmp_sign — Sinal do número

Descrição

int gmp_sign ( resource $a )

Verifica o sinal de um número.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Retorna 1 se a é positivo, -1 se a é negativo, e 0 se a for zero.

Exemplos

Exemplo #1 Exemplo da gmp_sign()

<?php
// positivo
echo gmp_sign("500") . "\n";
       
// negativo
echo gmp_sign("-500") . "\n";
       
// zero
echo gmp_sign("0") . "\n";
?>

O exemplo acima irá imprimir:

1
-1
0
