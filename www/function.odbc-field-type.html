odbc_field_type

(PHP 4, PHP 5)

odbc_field_type — Tipo de dados de um campo

Descrição

string odbc_field_type ( resource $result_id , int $field_number )

Obtém o tipo SQL do campo referenciado pelo número num dado identificador de resultado.

Parâmetros

result_id

O identificador de resultado.

field_number

O número do campo. Campos numerados a partir de 1.

Valor Retornado

Retorna o tipo do campo como uma string, ou FALSE em erro.
