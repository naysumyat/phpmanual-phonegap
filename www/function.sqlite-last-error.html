sqlite_last_error

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_last_error — Retorna o código de erro para o último erro em um banco de dados

Descrição

int sqlite_last_error ( resource $dbhandle )

Retorna o código de erro da última operação realizada no banco de dados db. Uma descrição do código de erro pode ser obtida usando-se a função sqlite_error_string().

Veja também sqlite_error_string().
