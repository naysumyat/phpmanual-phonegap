XMLWriter::openURI

xmlwriter_open_uri

(PHP 5 >= 5.1.2, PECL xmlwriter >= 0.1.0)

XMLWriter::openURI -- xmlwriter_open_uri — Create new xmlwriter using source uri for output

Descrição

Estilo orientado à objeto

bool XMLWriter::openURI ( string $uri )

Estilo procedural

resource xmlwriter_open_uri ( string $uri )

Creates a new XMLWriter using uri for the output.

Parâmetros

uri

The URI of the resource for the output.

Valor Retornado

Estilo orientado à objeto: Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Estilo procedural: Returns a new xmlwriter recurso for later use with the xmlwriter functions on success, FALSE on error.
