ctype_upper

(PHP 4 >= 4.0.4, PHP 5)

ctype_upper — Verifica se os caracteres são maiúsculos

Descrição

bool ctype_upper ( string $text )

Verifica se todos os caracteres na string fornecida, text, são maiúsculos.

Parâmetros

text

A string a ser testada.

Valor Retornado

Retorna TRUE se todos caracteres em text é uma letra maiúscula no locale atual.

Exemplos

Exemplo #1 Um exemplo da ctype_upper() (usando o locale padrão)

<?php
$strings = array('AKLWC139', 'LMNSDO', 'akwSKWsm');
foreach ($strings as $testcase) {
    if (ctype_upper($testcase)) {
        echo "The string $testcase consists of all uppercase letters.\n";
    } else {
        echo "The string $testcase does not consist of all uppercase letters.\n";
    }
}
?>

O exemplo acima irá imprimir:

The string AKLWC139 does not consist of all uppercase letters.
The string LMNSDO consists of all uppercase letters.
The string akwSKWsm does not consist of all uppercase letters.
