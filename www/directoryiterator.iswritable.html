DirectoryIterator::isWritable

(PHP 5)

DirectoryIterator::isWritable — Retorna true se o arquivo pode ser modificado

Descrição

bool DirectoryIterator::isWritable ( void )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Verifica se a entrada pode ser modificada.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

TRUE se o arquivo pode ser escrito, FALSE caso contrário
