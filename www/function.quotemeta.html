quotemeta

(PHP 4, PHP 5)

quotemeta — Adiciona uma barra invertida antes dos meta caracteres

Descrição

string quotemeta ( string $str )

Retorna uma versão de str com uma barra invertida (\) antes de cada um destes caracteres:

. \ + * ? [ ^ ] ( $ )

Parâmetros

str

A string de entrada.

Valor Retornado

Retorna a string com meta caracteres com escape.

Notas

Nota: Esta função é binary-safe.
