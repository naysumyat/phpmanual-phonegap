is_nan

(PHP 4 >= 4.2.0, PHP 5)

is_nan — Verifica se um valor não é um número

Descrição

bool is_nan ( float $val )

Verifica se val é 'não é um número', como o resultado de acos(1.01).

Parâmetros

val

The value to check

Valor Retornado

Retorna TRUE se val é 'não é um numéro', caso contrário FALSE.
