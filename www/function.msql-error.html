msql_error

(PHP 4, PHP 5)

msql_error — Retorna a mensagem de erro da última chamada a msql

Descrição

string msql_error ( void )

msql_error() retorna o último erro emitido pelo servidor mSQL. Note que somente a última mensagem de erro é acessível com msql_error().

Valor Retornado

A última mensagem de erro ou uma string vazia se nenhum erro foi emitido.
