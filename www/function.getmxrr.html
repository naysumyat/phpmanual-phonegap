getmxrr

(PHP 4, PHP 5)

getmxrr — Obtém registros MX correspondendo ao nome do host de Internet fornecido.

Descrição

bool getmxrr ( string $hostname , array &$mxhosts [, array &$weight ] )

Pesquisa DNS para registros MX correspondendo a hostname.

Parâmetros

hostname

The Internet host name.

mxhosts

Uma lista dos registros MX encontrados é colocada dentro de um array mxhosts.

weight

Se o array weight é dado, ele será preenchido com a informação de weight colhida.

Valor Retornado

Retorna TRUE se algum registro foi encontrado; retorna FALSE se nenhum registro foi encontrado ou se um erro ocorreu.

Notas

Nota:

Esta função não seria usada para propósitos de verificação de endereços. Apenas os mailexchangers encontrados no DNS são retornados, entretanto, de acordo com » RFC 2821 quando nenhum mail exchangers está listado, hostname seria usado como apenas mail exchanger com uma prioridade 0.

Nota:

Esta função não é implementada em plataformas Windows. Tente a classe » PEAR » Net_DNS.
