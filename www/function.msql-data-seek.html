msql_data_seek

(PHP 4, PHP 5)

msql_data_seek — Move o ponteiro interno dos registros

Descrição

bool msql_data_seek ( resource $result , int $row_number )

msql_data_seek() move o ponteiro interno de registro para o conjunto de resultados identificado por query_identifier para o número do registro indicado. A próxima chamada a msql_fetch_row() irá retornar este registro.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

row_number

A linha desejada.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

  • msql_fetch_array() - Obtém um registro como uma matriz
  • msql_fetch_object() - Obtem uma linha como um objeto
  • msql_fetch_row() - Retorna uma linha como uma matriz numerada
  • msql_result() - Retorna dados do resultado
