sqlite_last_insert_rowid

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_last_insert_rowid — Retorna o rowid da ultima linha inserida.

Descrição

int sqlite_last_insert_rowid ( resource $dbhandle )

Retorna o rowid da ultima linha inserida no banco de dados dbhandle, se ele foi criado com um campo de auto-incremento.

Dica

Você pode criar campos de auto-incremento no SQLite declarando eles como INTEGER PRIMARY KEY ao criar a sua tabela.
