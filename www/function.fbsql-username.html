fbsql_username

(PHP 4 >= 4.0.6, PHP 5)

fbsql_username — Get or set the username for the connection

Descrição

string fbsql_username ( resource $link_identifier [, string $username ] )

Get or set the username used for the connection.

Parâmetros

link_identifier

Um identificador de conexão FrontBase retornado por fbsql_connect() ou fbsql_pconnect().

Se opcional e não especificado, a função tentará encontrar uma conexão aberta para o servidor FrontBase server e se não for encontrada uma conexão dessa ela irá tentar criar uma como se fbsql_connect() fosse chamada sem argumentos

username

If provided, this is the new username to set.

Valor Retornado

Returns the current username used with the connection, as a string.
