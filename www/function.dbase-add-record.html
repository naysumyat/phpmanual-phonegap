dbase_add_record

(PHP 4, PHP 5)

dbase_add_record — Adiciona um registro ao banco de dados

Descrição

bool dbase_add_record ( int $dbase_identifier , array $record )

Adiciona os dados so banco de dados.

Parâmetros

dbase_identifier

O identificador do banco de dados, retornado por dbase_open() ou dbase_create().

record

Uma matriz indexada de dados. O número de itens deve ser igual ao número de campos no banco de dados, se não dbase_add_record() irá falhar.

Nota:

Se você esta usando o valor de retorno de dbase_get_record() para este parâmetro, lembre-se de reiniciar o valor de deleted.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Exemplo #1 Inserindo um registro num banco de dados dBase

<?php

// open in read-write mode
$db = dbase_open('/tmp/test.dbf', 2);

if ($db) {
  dbase_add_record($db, array(
      date('Ymd'),
      'Maxim Topolov',
      '23',
      'max@example.com',
      'T'));
  dbase_close($db);
}

?>
