ArrayObject::__construct

(PHP 5 >= 5.0.0)

ArrayObject::__construct — Constrói um novo objeto array

Descrição

ArrayObject::__construct() ( mixed $input )

Constrói um novo object array.

Parâmetros

input

O parâmetro input aceita um array ou um outro ArrayObject.

Valor Retornado

Não há valor retornado.

Exemplos

Exemplo #1 Exemplo de ArrayObject::__construct()

<?php
$array = array('1' => 'um',
               '2' => 'dois',
               '3' => 'três');

$arrayobject = new ArrayObject($array);

var_dump($arrayobject);
?>

O exemplo acima irá imprimir:

object(ArrayObject)#1 (3) {
  [1]=>
  string(2) "um"
  [2]=>
  string(4) "dois"
  [3]=>
  string(4) "três"
}
