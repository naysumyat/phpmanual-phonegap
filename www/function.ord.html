ord

(PHP 4, PHP 5)

ord — Retorna o valor ASCII do caractere

Descrição

int ord ( string $string )

Retorna o valor ASCII do primeiro caractere de string.

Esta função complementa chr().

Parâmetros

string

Um caractere.

Valor Retornado

Retorna o valor ASCII como um inteiro.

Exemplos

Exemplo #1 Exemplo da ord()

<?php
$str = "\n";
if (ord($str) == 10) {
    echo "O primeiro caractere de \$str é uma nova linha.\n";
}
?>
