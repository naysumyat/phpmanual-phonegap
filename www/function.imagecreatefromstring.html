imagecreatefromstring

(PHP 4 >= 4.0.4, PHP 5)

imagecreatefromstring — Cria uma nova imagem a apartir da imagem na string

Descrição

resource imagecreatefromstring ( string $image )

imagecreatefromstring() retorna um identificador de imagem representando a imagem obtida a apartir da string dada.
