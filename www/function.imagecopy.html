imagecopy

(PHP 4, PHP 5)

imagecopy — Copia parte de uma imagem

Descrição

int imagecopy ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $src_w , int $src_h )

Copia parte de src_im em dst_im começando nas coordenadas x,y src_x, src_y com a largura de src_w e altura de src_h. A porção definida será copiada nas coordenadas x,y, dst_x e dst_y.
