mssql_free_statement

(PHP 4 >= 4.3.2, PHP 5, PECL odbtp >= 1.1.1)

mssql_free_statement — Libera comando da memória

Descrição

bool mssql_free_statement ( resource $stmt )

mssql_free_statement() somente precisa ser chamada se você está preocupado usando muita memória quando seu script é executado. Toda memória de comandos será automaticamente liberada quando o script termina. Você pode chamhar mssql_free_statement() com o identificador do comando como um argumento e o comando associado na memória será liberado.

Parâmetros

stmt

Resource do comando, obtido com mssql_init().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
