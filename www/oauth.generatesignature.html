OAuth::generateSignature — Generate a signature

Descrição

public string OAuth::generateSignature ( string $http_method , string $url [, mixed $extra_parameters ] )

Generate a signature based on the final HTTP method, URL and a string/array of parameters.

Parâmetros

http_method

HTTP method for request

url

URL for request

extra_parameters

String or array of additional parameters.

Valor Retornado

A string containing the generated signature ou FALSE em caso de falha
