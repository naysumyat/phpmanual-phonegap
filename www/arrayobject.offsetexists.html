ArrayObject::offsetExists

(PHP 5 >= 5.0.0)

ArrayObject::offsetExists — Retorna se o índice solicitado existe ou não

Descrição

bool ArrayObject::offsetExists ( mixed $index )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Parâmetros

index

O índice a ser verificado.

Valor Retornado

TRUE se o requisitado $index existe, FALSE caso contrário
