Configurações em Execução

O comportamento dessas funções podem ser modificado pelas configurações do php.ini.

Opções de Configuração de Variáveis
Nome Valor Padrão Alterável Changelog
unserialize_callback_func NULL PHP_INI_ALL Disponível desde o PHP 4.2.0.
Para mais detalhes e definições dos modos PHP_INI_*, veja Aonde uma configuração deve ser definida.

Here's a short explanation of the configuration directives.

unserialize_callback_func string

A função unserialize() de callback será chamada (com nome de classe indefinido como parâmetro), se o deserializador encontra uma classe indefinda que deve ser instanciada. Um aviso aparece se a função especificada não estiver definida, ou se a função não inclue/implementa a classe que falta. Então apenas configure essa entrada se você realmente quer implementa uma função de callback.

Veja também unserialize().
