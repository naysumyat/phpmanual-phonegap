sqlite_prev

SQLiteResult::prev

(PHP 5 < 5.4.0)

sqlite_prev -- SQLiteResult::prev — Seek to the previous row number of a result set

Descrição

bool sqlite_prev ( resource $result )

Estilo orientado à objeto (method):

bool SQLiteResult::prev ( void )

sqlite_prev() seeks back the result handle to the previous row.

Parâmetros

result

The SQLite result resource. This parameter is not required when using the object-oriented method.

Nota:

Esta função não pode ser utilizada com manipuladores de resultados sem buffer.

Valor Retornado

Returns TRUE on success, or FALSE if there are no more previous rows.

Veja Também

  • sqlite_has_prev() - Returns whether or not a previous row is available
  • sqlite_rewind() - Vai para a primeira linha.
  • sqlite_next() - Move para a próxima linha.
