getmyinode

(PHP 4, PHP 5)

getmyinode — Obtém o inode do script atual

Descrição

int getmyinode ( void )

Obtém o inode do script atual.

Valor Retornado

Retorna o inode do atual script como um inteiro, ou FALSE em erro.

Notas

Nota: esta função não é implementada na plataforma Windows
