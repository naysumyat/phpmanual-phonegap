fam_cancel_monitor

(PHP 5 <= 5.0.5)

fam_cancel_monitor — Finalizando monitoramento

Descrição

bool fam_cancel_monitor ( resource $fam , resource $fam_monitor )

Finaliza um monitoramento no recurso.

Além disso um evento FAMAcknowledge ocorre.

Parâmetros

fam

Um resource representando uma conexão para um serviço FAM retornado pela fam_open()

fam_monitor

Um resource retornado por uma das funções fam_monitor_XXX

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Veja Também

  • fam_monitor_file() - Monitora alterações em um arquivo
  • fam_monitor_directory() - Monitora alterações em diretórios
  • fam_monitor_collection() - Monitora alterações de uma coleção de arquivos em um diretório
  • fam_suspend_monitor() - Suspende monitoramento temporariamente
