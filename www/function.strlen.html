strlen

(PHP 4, PHP 5)

strlen — Retorna o tamanho de uma string

Descrição

int strlen ( string $string )

Retorna o tamanho da dada string.

Parâmetros

string

A string a ter seu tamanho contado.

Valor Retornado

O tamanho da string em sucesso, e 0 se a string é vazia.

Exemplos

Exemplo #1 Exemplo da strlen()

<?php
$str = 'abcdef';
echo strlen($str); // 6

$str = ' ab cd ';
echo strlen($str); // 7
?>
