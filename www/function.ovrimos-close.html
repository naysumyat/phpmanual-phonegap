ovrimos_close

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_close — Fecha o conexão com o ovrimos

Descrição

void ovrimos_close ( int $connection )

Fechar a especificada connection com o Ovrimos. Isto tem o efeito de desfazer as transações que não foram gravadas.

Parâmetros

connection

O identificador de conexão do Ovrimos, retornado pela ovrimos_connect().

Valor Retornado

Não há valor retornado.
