dbase_numrecords

(PHP 4, PHP 5)

dbase_numrecords — Obtém o número de registros no banco de dados

Descrição

int dbase_numrecords ( int $dbase_identifier )

Obtém o número de registros (linhas) no banco de dados especificado.

Nota:

Número de registros são entre 1 e dbase_numrecords($db), enquanto que números de campo são entre 0 e dbase_numfields($db)-1.

Parâmetros

dbase_identifier

O identificador do banco de dados, retornado por dbase_open() ou dbase_create().

Valor Retornado

O número de registros no banco de dados, ou FALSE se ocorrer um erro.

Exemplos

Exemplo #1 Lendo todos os registros de um banco de dados

<?php

// open in read-only mode
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
  $record_numbers = dbase_numrecords($db);
  for ($i = 1; $i <= $record_numbers; $i++) {
      // do something here, for each record
  }
}

?>
