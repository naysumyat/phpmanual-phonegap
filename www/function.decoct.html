decoct

(PHP 4, PHP 5)

decoct — Decimal para octal

Descrição

string decoct ( int $number )

Retorna uma string contendo a representação octal do parâmetro number. O maior número que pode ser convertido é 4294967295 em decimal, resultando em "37777777777".

Parâmetros

number

Valor decimal para converter

Valor Retornado

String octal representando o number

Exemplos

Exemplo #1 Exemplo da decoct()

<?php
echo decoct(15) . "\n";
echo decoct(264);
?>

O exemplo acima irá imprimir:

17
410
