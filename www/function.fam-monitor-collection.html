fam_monitor_collection

(PHP 5 <= 5.0.5)

fam_monitor_collection — Monitora alterações de uma coleção de arquivos em um diretório

Descrição

resource fam_monitor_collection ( resource $fam , string $dirname , int $depth , string $mask )

Requisita monitoramento para uma coleção de arquivos dentro de um diretório.

Um evento FAM irá ser gerado quando o status dos arquivos forem alterados. Os possíveis códigos do evento são descritos em detalhes na parte de constantes dessa seção.

Parâmetros

fam

Um resource representando uma conexão para o serviço FAM retornado pela fam_open()

dirname

Caminho do diretório para monitorar arquivos

depth

O máximo de busca depth iniciando deste diretório

mask

Um padrão shell mask restringido os nomes de arquivos parar serem verificados

Valor Retornado

Retorna um resource de monitoramento ou FALSE em erros.
