xml_error_string

(PHP 4, PHP 5)

xml_error_string — Obtém uma string de erro do analisador XML

Descrição

string xml_error_string ( int $code )

Obtém a string do erro do analisador XML associado com um dado code.

Parâmetros

code

Um código de erro de xml_get_error_code().

Valor Retornado

Retorna uma string com uma descrição textual do código de erro code, ou FALSE se nenhuma descrição for encontrada.
