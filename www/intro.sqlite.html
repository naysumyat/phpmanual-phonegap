Introdução

Esta é uma extensão para a Ferramenta de Banco de Dados Embutível SQL. SQLite é uma biblioteca em C que implementa um banco de dados embutível SQL. Programas que são ligados com a biblioteca SQLite podem ter acesso SQL a um banco de dados SQL sem executar um processo RDBMS separado.

SQLite não é uma biblioteca de cliente usado para conectar em um grande servidor de banco de dados. SQLite é o servidor. A biblioteca SQLite lê e escreve diretamente para e de os arquivos do banco de dados no disco.

Nota:

Para maiores informações veja o website do SQLite (» http://sqlite.org/).
