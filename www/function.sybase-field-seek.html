sybase_field_seek

(PHP 4, PHP 5)

sybase_field_seek — Define o índice do campo

Descrição

bool sybase_field_seek ( resource $result , int $field_offset )

Muda para o índice de campo especificado. Se a próxima chamada a sybase_fetch_field() não incluir um índice de campo, este campo será retornado.

Parâmetros

result

field_offset

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
