filepro_fieldcount

(PHP 4, PHP 5 <= 5.1.6)

filepro_fieldcount — Conta quantos campos tem em um banco de dados filePro

Descrição

int filepro_fieldcount ( void )

Retorna o número de campos (colunas) em um banco de dados filePro aberto.

Valor Retornado

Retorna o número de campos no banco de dados filePro aberto, ou FALSE em caso de erro.
