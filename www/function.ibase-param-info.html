ibase_param_info

(PHP 5)

ibase_param_info — Retorna informação sobre um parâmetro em uma consulta preparada

Descrição

array ibase_param_info ( resource $query , int $param_number )

Retorna uma matriz com informações sobre um parâmetro após a consulta ter sido preparada.

Parâmetros

query

Um manipulador de query preparada do InterBase.

param_number

O índice do parâmetro.

Valor Retornado

Retorna um array com as seguintes chaves: name, alias, relation, length and type.
