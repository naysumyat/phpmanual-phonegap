ovrimos_fetch_into

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_fetch_into — Obtem uma linha de um conjunto de resultados

Descrição

bool ovrimos_fetch_into ( int $result_id , array $result_array [, string $how [, int $rownumber ]] )

ovrimos_fetch_into() obtém uma linha do conjunto de resultados em result_array, a qual deve ser passada por referencia. A linha a ser obtida é determinada pelos dois últimos parâmetros. how é um de Next (padrão), Prev, First, Last, Absolute, corespondendo ao próximo da posição atual, a anterior a posição atual, o primeiro resultado, aúltima posição e uma posição absoluta a partir do início (essencialmente equivalente a 'first' mas precisa de 'rownumber'). Não é importante a diferença entre maiúsculas e minúsculas. rownumber é opcional exceto para posicionamento absoluto. Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplo #1 Um exemplo

<?php
$conn=ovrimos_connect("neptune", "8001", "admin", "password");
if ($conn!=0) {
    echo "Connection ok!";
    $res=ovrimos_exec($conn, "select table_id, table_name from sys.tables");
    if ($res != 0) {
        echo "Statement ok!";
        if (ovrimos_fetch_into($res, &$row)) {
            list($table_id, $table_name) = $row;
            echo "table_id=" . $table_id . ", table_name=" . $table_name . "\n";
            if (ovrimos_fetch_into($res, &$row)) {
                list($table_id, $table_name) = $row;
                echo "table_id=" . $table_id . ", table_name=" . $table_name . "\n";
            } else {
                echo "Next: error\n";
            }
        } else {
            echo "First: error\n";
        }
        ovrimos_free_result($res);
    }
    ovrimos_close($conn);
}
?>
Este exemplo irá obter uma linha.
