imagecopymerge

(PHP 4 >= 4.0.1, PHP 5)

imagecopymerge — Copia e combina uma parte de uma imagem

Descrição

int imagecopymerge ( resource $dst_im , resource $src_im , int $dst_x , int $dst_y , int $src_x , int $src_y , int $src_w , int $src_h , int $pct )

Copia parte de src_im em dst_im começando nas coordenadas src_x, src_y com a largura de src_w e altura de src_h. A porção definida será copiada nas coordenadas x,y, dst_x e dst_y. As duas imagens serão combinadas de acordo com pct o qual pode variar de 0 a 100. Quando pct = 0, não haverá modificação, quando 100 esta função funciona de modo identico a imagecopy() para imagens de paleta, enquanto implemente transparencia alpha para imagens true colour.

Nota:

Esta função foi adicionada no PHP 4.0.6
