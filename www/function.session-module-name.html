session_module_name

(PHP 4, PHP 5)

session_module_name — Obtém e/ou define o módulo da sessão atual

Descrição

string session_module_name ([ string $module ] )

session_module_name() obtém o nome do módulo da sessão atual.

Parâmetros

module

Se module é especificado, este módulo será usado.

Valor Retornado

Retorna o do módulo da sessão atual.
