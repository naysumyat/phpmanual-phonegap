mssql_num_rows

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_num_rows — Retorna o número de linhas em um resultado

Descrição

int mssql_num_rows ( resource $result )

mssql_num_rows() retorna o número de linhas em um conjunto de resultados.

Parâmetros

result

O resource do resultado que está sendo avaliado. Este resultado vem de uma chamada a mssql_query().

Valor Retornado

Retorna o número de linhas, como um inteiro.

  • mssql_query() - Envia uma query ao MS SQL
  • mssql_fetch_row() - Retorna uma matriz enumerada
