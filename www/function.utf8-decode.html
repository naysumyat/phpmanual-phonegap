utf8_decode

(PHP 4, PHP 5)

utf8_decode — Converte uma string com caracteres ISO-8859-1 codificadas com UTF-8 para single-byte ISO-8859-1.

Descrição

string utf8_decode ( string $data )

Esta função decodifica data, assumido ser codificada em UTF-8, para ISO-8859-1.

Parâmetros

data

A string codificada UTF-8.

Valor Retornado

Retorna a tradução ISO-8859-1 de data.
