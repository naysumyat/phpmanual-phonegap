quoted_printable_decode

(PHP 4, PHP 5)

quoted_printable_decode — Converte uma string quoted-printable para uma string de 8 bit

Descrição

string quoted_printable_decode ( string $str )

Esta função retorna uma string binária de 8-bit correspondendo a string quoted printable decodificada (de acordo com » RFC2045, seção 6.7, não » RFC2821, seção 4.5.2, adicionais periodos não são removidos do início da linha).

Esta função é similar a imap_qprint(), exceto que esta não requer o módulo IMAP para funcionar.

Parâmetros

str

A string de entrada.

Valor Retornado

Retorna uma string binária de 8-bit.
