msql_list_tables

(PHP 4, PHP 5)

msql_list_tables — Lista as tabelas em um banco de dados mSQL

Descrição

resource msql_list_tables ( string $database [, resource $link_identifier ] )

msql_list_tables() lista as tabelas no database(banco de dados) especificado . Ela retorna um conjunto de resultados que pode ser utilizado com quaisquer funções que obtém resultados de um conjunto de resultados, como, por exemplo, msql_fetch_array().

Esta função retorna FALSE em caso de falha.

Para compatibilidade com versões anteriores, o apelido chamado msql_listtables() pode ser usado. Esta, entretando, esta obsoleta e não é recomendada.
