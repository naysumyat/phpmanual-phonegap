hebrevc

(PHP 4, PHP 5)

hebrevc — Converte um texto lógico Hebráico para um texto visual com conversão newline

Descrição

string hebrevc ( string $hebrew_text [, int $max_chars_per_line ] )

É similar a hebrev() com a diferença de que ela converte newlines (\n) para "<br>\n".

A função tenta evitar quebra de palavras.

Parâmetros

hebrew_text

Uma string de entrada em Hebraico.

max_chars_per_line

Este parâmetro opcional indica o número máximo de caracteres por linha que será retornada.

Valor Retornado

Retorna a string visual.
