bcsub

(PHP 4, PHP 5)

bcsub — Subtrai um número de precisão arbitrária de outro

Descrição

string bcsub ( string $left_operand , string $right_operand [, int $scale ] )

Subtrair o left_operand do right_operand.

Parâmetros

left_operand

O operando da esquerda, como uma string.

right_operand

O operando da direita, como uma string.

scale

Este parâmetro opcional é usado para definir o número de digitos depois do ponto decimal no resultado. Você pode definir a escala padrão global para todas as funções usando bcscale().

Valor Retornado

O resultado da subtração, como uma string.

Exemplos

Exemplo #1 Exemplo de bcsub()

<?php

$a = '1.234';
$b = '5';

echo bcsub($a, $b);     // -3
echo bcsub($a, $b, 4);  // -3.7660

?>
