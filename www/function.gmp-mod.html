gmp_mod

(PHP 4 >= 4.0.4, PHP 5)

gmp_mod — Operação Módulo

Descrição

resource gmp_mod ( resource $n , resource $d )

Calcula n módulo d. O resultado é sempre positivo, o sinal de d é ignorado.

Parâmetros

n

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

d

O módulo que será calculado.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_mod()

<?php
$mod = gmp_mod("8", "3");
echo gmp_strval($mod) . "\n";
?>
