dbase_numfields

(PHP 4, PHP 5)

dbase_numfields — Obtém o número de campos em um banco de dados

Descrição

int dbase_numfields ( int $dbase_identifier )

Obtém o número de campos(colunas) no banco de dados especificados.

Nota:

Números de campo estão entre 0 e dbase_numfields($db)-1, enquanto que números de registro estão entre 1 e dbase_numrecords($db).

Parâmetros

dbase_identifier

O identificador do banco de dados, retornado por dbase_open() ou dbase_create().

Valor Retornado

O números de campos em um banco de dados, ou FALSE se acontecer um erro.

Exemplos

Exemplo #1 Exemplo dbase_numfields()

<?php

$rec = dbase_get_record($db, $recno);
$nf  = dbase_numfields($db);
for ($i = 0; $i < $nf; $i++) {
  echo $rec[$i], "\n";
}

?>
