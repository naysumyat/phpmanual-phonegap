min

(PHP 4, PHP 5)

min — Encontra o menor valor

Descrição

mixed min ( array $values )
mixed min ( mixed $value1 , mixed $value2 [, mixed $value3... ] )

Se o primeiro e único argumento é um array, min() retorna o menor valor no array. Se no mínimo dois parâmetros são fornecidos, min() retornará o menor deles.

Nota:

O PHP irá avaliar uma string não numérica como 0 se comparado a integer, mas ainda retorna a string como se ela fosse numericamente o menor valor. Se vários agumentos forem avaliados como 0, min() irá retornar o menor valor alfanumericamente se alguma string é dada, senão um número 0 é retornado.

Parâmetros

values

Um array contendo os valores.

Valor Retornado

min() retorna o menor numericalmente dos valores dos parâmetros.

Exemplos

Exemplo #1 Exemplos de min()

<?php
echo min(2, 3, 1, 6, 7);  // 1
echo min(array(2, 4, 5)); // 2

echo min(0, 'hello');     // 0
echo min('hello', 0);     // hello
echo min('hello', -1);    // -1

// Com arrays múltiplos, min compara da esquerda para direita,
// assim nesse exemplo: 2 == 2, mas 4 < 5
$val = min(array(2, 4, 8), array(2, 5, 1)); // array(2, 4, 8)

// Se ambos forem um array e um não array, o array
// nunca será retornado porque ele é sempre considerado o maior
$val = min('string', array(2, 5, 7), 42);   // string
?>
