ovrimos_cursor

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_cursor — Retorna o nome do cursor

Descrição

string ovrimos_cursor ( int $result_id )

Obtém o nome do cursor. Útil quando você quer fazer updates ou deletes posicionados.

Parâmetros

result_id

Um identificador de resultado, retornado pela ovrimos_execute() ou ovrimos_exec().

Valor Retornado

Retorna o nome como uma string, ou FALSE em erro.
