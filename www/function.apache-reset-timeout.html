apache_reset_timeout

(PHP 5 >= 5.1.0)

apache_reset_timeout — Reinicializa o timer de escrita do Apache

Descrição

bool apache_reset_timeout ( void )

apache_reset_timeout() reinicializa o timer de escrita do Apache, que tem o valor padrão de 300 segundo. Com set_time_limit(0); ignore_user_abort(true) e chamadas periódicas de apache_reset_timeout(), o Apache pode, teoricamente, executar para sempre.

Essa função requer Apache 1.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Notas

Nota: This function is disabled when PHP is running in safe mode.

Veja Também

  • set_time_limit() - Limita o tempo de execução do script
  • ignore_user_abort() - Estabelece se acontecerá o encerramento do script quando o usuário abortar a conexão
