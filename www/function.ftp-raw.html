ftp_raw

(PHP 5)

ftp_raw — Envia um comando arbritário para o servidor FTP

Descrição

array ftp_raw ( resource $ftp_stream , string $command )

Envia um command arbritário para o servidor FTP.

Parâmetros

ftp_stream

O identificador da conexão FTP.

command

O comando a executar.

Valor Retornado

Retorna a resposta do servidor como um array de strings. Nenhuma interpretação é realizada na string de resposta, nem ftp_raw() determina se o comando foi bem sucedido

Exemplos

Exemplo #1 Usando ftp_raw() para fazer o login em um servidor FTP manualmente.

<?php
$fp = ftp_connect("ftp.example.com");

/* This is the same as: 
   ftp_login($fp, "joeblow", "secret"); */
ftp_raw($fp, "USER joeblow");
ftp_raw($fp, "PASS secret");
?>
