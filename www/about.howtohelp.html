Como ajudar a melhoras a documentação

Há três maneiras de ajudar a melhoras a documentação.

Se um erro é encontrado no manual, em qualquer linguagem, favor reporte-nos usando o sistema de bug em » http://bugs.php.net/. Classifique o bug "Documentation Problem". Todos os problemas relacionados com a documentação, inclusive sobre os formatos de manuais, deverão ser apresentados em relatórios de bugs.

Nota:

Por favor não abuse do sistema de bugs enviado pedidos de ajuda. Neste caso, use uma das muitas opções » opções de suporte.

Ao contribuir com notas, os usuários podem fornecer exemplos adicionais, e esclarecimento para outros leitores. Mas por favor, não envie reporte de bug usando o sistema de anotações. Para mais detalhes, veja a seção intitulada 'Sobre as notas de usuários'.

O manual PHP é traduzindo na maiorias da línguas. Sabendo Inglês e uma língua estrangeira permite uma maneira de ajudar a melhorar o manual PHP trabalhando com um time de tradução. Para mais informações sobre como iniciar uma nova tradução, ou ajudar no projeto de tradução atual, por favor leia » https://wiki.php.net/doc/howto.

O Projeto de documentação do PHP também possui um canal IRC onde autores se reunem. No #php.doc em irc.efnet.org e discutem maneiras de melhorar a documentação PHP.
