ibase_free_result

(PHP 5)

ibase_free_result — Libera um conjunto de resultados

Descrição

bool ibase_free_result ( resource $result_identifier )

Libera um conjunto de resultado.

Parâmetros

result_identifier

Um conjunto de resultado criado pela ibase_query() ou ibase_execute().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
