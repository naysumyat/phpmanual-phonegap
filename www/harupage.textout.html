HaruPage::textOut

(PECL haru >= 0.0.1)

HaruPage::textOut — Print the text on the specified position

Descrição

bool HaruPage::textOut ( float $x , float $y , string $text )

Prints the text on the specified position.

Parâmetros

x

y

text

Valor Retornado

Returns TRUE on success.

Erros

Joga um HaruException em erro.
