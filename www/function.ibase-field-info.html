ibase_field_info

(PHP 5)

ibase_field_info — Obtém informação sobre um campo

Descrição

array ibase_field_info ( resource $result , int $field_number )

Retorna uma matriz com informações sobre um campo após uma consulta SELECT ter sido executada.

Parâmetros

result

Um identificador de resultado do InterBase.

field_number

Índice do campo.

Valor Retornado

Retorna um array com as seguintes chaves: name, alias, relation, length e type.

Exemplos

Exemplo #1 Exemplo da ibase_field_info()

<?php
    $rs = ibase_query("SELECT * FROM tablename");
    $coln = ibase_num_fields($rs);
    for ($i = 0; $i < $coln; $i++) {
        $col_info = ibase_field_info($rs, $i);
        echo "name: ". $col_info['name']. "\n";
        echo "alias: ". $col_info['alias']. "\n";
        echo "relation: ". $col_info['relation']. "\n";
        echo "length: ". $col_info['length']. "\n";
        echo "type: ". $col_info['type']. "\n";
    }
?>
