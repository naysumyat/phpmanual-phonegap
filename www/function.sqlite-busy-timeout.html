sqlite_busy_timeout

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_busy_timeout — Define o tempo de espera quando o banco de dados estiver ocupado

Descrição

void sqlite_busy_timeout ( resource $dbhandle , int $milésimos )

Define o limite de tempo que o sqlite irá esperar para um banco de dados (dbhandle) se tornar disponível para uso em milésimos. Se o parâmetro milésimos for 0, os gerenciadores de ocupado serão desabilitados e o sqlite irá retornar imediatamente com um código SQLITE_BUSY se outro processo/thread tiver o banco de dados travado para uma atualização.

O PHP define o tempo padrão de espera em 60 segundos quando o banco de dados é aberto.

Nota:

Existem mil (1000) milésimos em um segundo.

Veja também sqlite_open().
