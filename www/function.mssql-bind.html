mssql_bind

(PHP 4 >= 4.0.7, PHP 5, PECL odbtp >= 1.1.1)

mssql_bind — Adiciona um parâmetro a um stored procedure ou a um remote stored procedure

Descrição

bool mssql_bind ( resource $stmt , string $param_name , mixed $var , int $type [, int $is_output [, int $is_null [, int $maxlen ]]] )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Veja também mssql_execute(), mssql_free_statement() e mssql_init().
