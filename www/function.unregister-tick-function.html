unregister_tick_function

(PHP 4 >= 4.0.3, PHP 5)

unregister_tick_function — Desregistra uma função executada a cada instante

Descrição

void unregister_tick_function ( string $function_name )

Desregistra uma função nomeada por function_name então isso não é mais executado quando uma tick é chamada.

Parâmetros

function_name

O nome da função, como string.

Valor Retornado

Não há valor retornado.
