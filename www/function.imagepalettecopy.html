imagepalettecopy

(PHP 4 >= 4.0.1, PHP 5)

imagepalettecopy — Copia a paleta de uma imagem para a outra

Descrição

void imagepalettecopy ( resource $destination , resource $source )

imagepalettecopy() copia a paleta da imagem de source para a imagem destination.

Parâmetros

destination

O resource da imagem de destino

source

O resource do fonte da imagem

Valor Retornado

Não há valor retornado.
