xml_get_current_line_number

(PHP 4, PHP 5)

xml_get_current_line_number — Obtém o número da linha para um analisador XML

Descrição

int xml_get_current_line_number ( resource $parser )

Obtém o número da linha atual para um dado analisador XML.

Parâmetros

parser

Uma referência ao analisador XML obter o número da linha.

Valor Retornado

Esta função retorna FALSE se parser não se refere a um analisador válido, senão ele retorna qual linha o analisador está atualmente em seu registro de dados (data buffer).
