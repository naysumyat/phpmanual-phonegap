filepro_fieldname

(PHP 4, PHP 5 <= 5.1.6)

filepro_fieldname — Obtém o nome de um campo

Descrição

string filepro_fieldname ( int $field_number )

Retorna o nome do campo correspondente a field_number.

Parâmetros

field_number

O número do campo.

Valor Retornado

Retorna o nome do campo como string, ou FALSE em caso de erro.
