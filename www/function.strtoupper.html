strtoupper

(PHP 4, PHP 5)

strtoupper — Converte uma string para maiúsculas

Descrição

string strtoupper ( string $string )

Retorna string com todos os caracteres do alfabeto convertidos para maiúsculas.

Note que os caracteres do alfabeto é determinado pelo local atual. Isto indica que no local padrão "C", caracteres como A com trema (ä) não serão convertidos.

Parâmetros

string

A string de entrada.

Valor Retornado

Retorna a string maiúscula.

Exemplos

Exemplo #1 Exemplo da strtoupper()

<?php
$str = "Mary Had A Little Lamb and She LOVED It So";
$str = strtoupper($str);
print $str; // Prints MARY HAD A LITTLE LAMB AND SHE LOVED IT SO
?>

Notas

Nota: Esta função é binary-safe.
