openssl_x509_export

(PHP 4 >= 4.2.0, PHP 5)

openssl_x509_export — Exports a certificate as a string

Descrição

bool openssl_x509_export ( mixed $x509 , string &$output [, bool $notext = TRUE ] )

openssl_x509_export() stores x509 into a string named by output in a PEM encoded format.

Parâmetros

x509

See Key/Certificate parameters for a list of valid values.

output

On success, this will hold the PEM.

notext

O parâmetro opcional notext afeta a exibição da saída; se ele é FALSE então informação legível para humano é incluida na saída. O valor padrão do notext é TRUE.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
