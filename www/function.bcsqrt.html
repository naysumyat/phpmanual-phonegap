bcsqrt

(PHP 4, PHP 5)

bcsqrt — Obtém a raiz quadrada de um número de precisão arbitrária

Descrição

string bcsqrt ( string $operand [, int $scale ] )

Retorna a raiz quadrada do operador.

Parâmetros

operand

O operando, como uma string.

scale

Este parâmetro opcional é usado para definir o número de digitos depois do ponto decimal no resultado. Você pode definir a escala padrão global para todas as funções usando bcscale().

Valor Retornado

Retorna a raiz quadrada como uma string, ou NULL se operand é negativo.

Exemplos

Exemplo #1 Exemplo de bcsqrt()

<?php

echo bcsqrt('2', 3); // 1.414

?>
