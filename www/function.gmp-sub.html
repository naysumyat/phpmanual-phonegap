gmp_sub

(PHP 4 >= 4.0.4, PHP 5)

gmp_sub — Subtrai números

Descrição

resource gmp_sub ( resource $a , resource $b )

Subtrai b de a e retorna o resultado.

Parâmetros

a

The number being subtracted from.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

b

The number subtracted from a.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_sub()

<?php
$sub = gmp_sub("281474976710656", "4294967296"); // 2^48 - 2^32
echo gmp_strval($sub) . "\n";
?>

O exemplo acima irá imprimir:

281470681743360
