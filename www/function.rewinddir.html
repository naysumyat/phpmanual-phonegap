rewinddir

(PHP 4, PHP 5)

rewinddir — Voltar (rewind) o manipulador de diretório

Descrição

void rewinddir ( resource $dir_handle )

Reinicia o stream do diretório indicado por dir_handle para o início do diretório.

Parâmetros

dir_handle

O manipulador de diretório resource aberto com opendir().
