DirectoryIterator::getExtension

(PHP 5 >= 5.3.6)

DirectoryIterator::getExtension — Retorna a extensão do arquivo do item corrente do DirectoryIterator

Descrição

public string DirectoryIterator::getExtension ( void )

Retorna a extensão do arquivo do item corrente do DirectoryIterator.

Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

A extensão do arquivo do item corrente do DirectoryIterator.

Exemplos

Exemplo #1 Exemplo do DirectoryIterator::getExtension()

Esse exemplo irá listar as extensões dos arquivos do diretório o qual o script irá rodar.

<?php
$dir = new DirectoryIterator(dirname(__FILE__));
foreach ($dir as $fileinfo) {
    if ($fileinfo->isFile()) {
        echo $fileinfo->getExtension() . "\n";
    }
}
?>

O exemplo acima irá imprimir algo similar à:
