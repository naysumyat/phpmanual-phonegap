Instalação

Tendo a libswf, tudo que você precisa fazer é configurar --with-swf[=DIR] onde DIR é o local contendo os diretórios include e lib. O diretório include deve conter o arquivo swf.h e o diretório lib deve conter o arquivo libswf.a. Se você descompactou a distribuição libswf os dois arquivos estarão em um diretório. Conseqüentemente você terá que copiar os arquivos para os locais correto manualmente.
