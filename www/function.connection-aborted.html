connection_aborted

(PHP 4, PHP 5)

connection_aborted — Verifica se o cliente está desconectado

Descrição

int connection_aborted ( void )

Verifica se o cliente está desconectado.

Valor Retornado

Retorna 1 se o cliente está desconectado, 0 caso contrário.
