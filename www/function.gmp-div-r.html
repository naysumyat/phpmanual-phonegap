gmp_div_r

(PHP 4 >= 4.0.4, PHP 5)

gmp_div_r — Resto da divisão de dois números

Descrição

resource gmp_div_r ( resource $n , resource $d [, int $round ] )

Calcula o resto da divisão inteira de n por d. O resto tem o sinal do argumento n, se não for zero.

Parâmetros

n

O número a ser dividido.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

d

O número o qual n será dividido.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

round

Veja a função gmp_div_q() para descrição do argumento round.

Valor Retornado

O resto, como um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_div_r()

<?php
$div = gmp_div_r("105", "20");
echo gmp_strval($div) . "\n";
?>
