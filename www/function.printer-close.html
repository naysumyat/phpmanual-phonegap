printer_close

(PECL printer SVN)

printer_close — Fecha uma conexão aberta com uma impressora

Descrição

void printer_close ( resource $handle )

Esta função fecha a conexão com a impressora. printer_close() também fecha o device context ativo.

handle deve ser um manipulador válido para uma impressora.

Exemplo #1 Exemplo printer_close()

<?php
$handle = printer_open();
printer_close($handle);
?>
