odbc_fetch_row

(PHP 4, PHP 5)

odbc_fetch_row — Obtém uma linha

Descrição

bool odbc_fetch_row ( resource $result_id [, int $row_number ] )

Obtém uma linha de dados que foi retornada pela odbc_do() ou odbc_exec(). Depois de odbc_fetch_row() ser chamada, os campos desta linha podem ser acessados com odbc_result().

Parâmetros

result_id

O identificador de resultado.

row_number

Se row_number não for especificado, odbc_fetch_row() irá tentar obter a próxima linha no conjunto de resultados. Chamadas para odbc_fetch_row() com ou sem row_number podem ser misturadas.

Para caminhar através do conjunto de resultados mais de uma vez, você pode usar odbc_fetch_row() com row_number 1, e então continuar fazendo odbc_fetch_row() sem row_number para ver os resultados. Se um driver não suportar a obtenção de linhas por números, o parâmetro row_number é ignorado.

Valor Retornado

Retorna TRUE se houver uma linha, FALSE caso contrário.
