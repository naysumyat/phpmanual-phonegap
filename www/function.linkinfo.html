linkinfo

(PHP 4, PHP 5)

linkinfo — Ler informações sobre um link

Descrição

int linkinfo ( string $path )

Obtém informação sobre um link.

Esta função é usada para verificar se um link (apontado por path) realmente existe (usando o mesmo método como a macro S_ISLNK definida em stat.h).

Parâmetros

path

Caminho para o link.

Valor Retornado

linkinfo() retorna o campo st_dev da estrutura stat do Unix C retornado pela chamada do sistema (system call) lstat. Retorna 0 ou FALSE em caso de erro.

Exemplos

Exemplo #1 Exemplo da linkinfo()

<?php

echo linkinfo('/vmlinuz'); // 835

?>

Notas

Nota: esta função não é implementada na plataforma Windows
