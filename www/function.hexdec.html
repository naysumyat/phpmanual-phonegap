hexdec

(PHP 4, PHP 5)

hexdec — Hexadecimal para decimal

Descrição

number hexdec ( string $hex_string )

Retorna o decimal equivalente do número hexadecimal representado pelo argumento hex_string. hexdec() converte uma string hexadecimal para um número decimal.

hexdec() irá ignorar qualquer caracter não hexadecimal que encontrar.

Parâmetros

hex_string

The hexadecimal string to convert

Valor Retornado

The decimal representation of hex_string

Changelog

Versão Descrição
Since 4.1.0 A função pode agora converter valores que são maior que o tipo integer da plataforma, retornará o valor como float neste caso.

Exemplos

Exemplo #1 exemplos hexdec()

<?php
var_dump(hexdec("See"));
var_dump(hexdec("ee"));
// ambos exibem "int(238)"

var_dump(hexdec("that")); // print "int(10)"
var_dump(hexdec("a0")); // print "int(160)"
?>
