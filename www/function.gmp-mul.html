gmp_mul

(PHP 4 >= 4.0.4, PHP 5)

gmp_mul — Multiplica números

Descrição

resource gmp_mul ( resource $a , resource $b )

Multiplica a por b e retorna o resultado.

Parâmetros

a

Um número que será multiplicado por b.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

b

Um número que será multiplicado por a.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_mul()

<?php
$mul = gmp_mul("12345678", "2000");
echo gmp_strval($mul) . "\n";
?>

O exemplo acima irá imprimir:

24691356000
