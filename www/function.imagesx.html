imagesx

(PHP 4, PHP 5)

imagesx — Obtem a largura da imagem

Descrição

int imagesx ( resource $image )

Retorna a largura do dado resource image.

Parâmetros

image

An image resource, returned by one of the image creation functions, such as imagecreatetruecolor().

Valor Retornado

Retorna a largura da image ou FALSE em erro.

Exemplos

Exemplo #1 Usando imagesx()

<?php

// cria uma imagem de 300*200
$img = imagecreatetruecolor(300, 200);

echo imagesx($img); // 300

?>

Veja Também

  • imagecreatetruecolor() - Cria uma nova imagem true color
  • getimagesize() - Obtém o tamanho de uma imagem
  • imagesy() - Obtém a altura da imagem
