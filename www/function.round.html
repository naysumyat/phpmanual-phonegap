round

(PHP 4, PHP 5)

round — Arredonda um número

Descrição

float round ( float $val [, int $precision ] )

Retornar um valor arredondado de val em precision casas decimais. precision pode ser negativo ou zero (padrão).

Nota: o PHP não manipula strings como "12,300.2" corretamente por padrão. Veja convertendo de strings.

Nota: O parâmetro precision foi acrescentado no PHP 4.

Parâmetros

val

O valor para arredondar

precision

O número opcional de dígitos decimais para arredondar, o padrão é 0

Valor Retornado

O valor arredondado

Exemplos

Exemplo #1 Exemplos da round()

<?php
echo round(3.4);         // 3
echo round(3.5);         // 4
echo round(3.6);         // 4
echo round(3.6, 0);      // 4
echo round(1.95583, 2);  // 1.96
echo round(1241757, -3); // 1242000
echo round(5.045, 2);    // 5.05
echo round(5.055, 2);    // 5.06
?>
