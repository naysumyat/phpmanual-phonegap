imagestringup

(PHP 4, PHP 5)

imagestringup — Desenha uma string verticalmente

Descrição

int imagestringup ( resource $image , int $font , int $x , int $y , string $s , int $col )

imagestringup() desenha a string s verticalmente na imagem identificada por image nas coordenadas x, y (em cima na esquerda é 0, 0) na cor col. Se a fonte for 1, 2, 3, 4 ou 5, uma fonte interna é usada.

Veja também imageloadfont().
