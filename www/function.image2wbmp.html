image2wbmp

(PHP 4 >= 4.0.5, PHP 5)

image2wbmp — Envia uma imagem para o browser ou arquivo

Descrição

bool image2wbmp ( resource $image [, string $filename [, int $threshold ]] )

image2wbmp() mostra ou salva uma versão WBMP de uma dada image.

Parâmetros

image

An image resource, returned by one of the image creation functions, such as imagecreatetruecolor().

filename

Caminho para salvar o arquivo. Se não informado, a imagem será mostrada diretamente.

threshold

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Exemplo #1 Exemplo image2wbmp()

<?php

$file = 'php.png';
$image = imagecreatefrompng($file);

header('Content-type: ' . image_type_to_mime_type(IMAGETYPE_WBMP));
image2wbmp($image); // output the stream directly

?>

Notas

Nota:

Suporte a WBMP esta disponível somente se o PHP foi compilado com a GD-1.8 ou posterior.
