ibase_blob_add

(PHP 5)

ibase_blob_add — Adiciona dados em um blob recém criado

Descrição

void ibase_blob_add ( resource $blob_handle , string $data )

ibase_blob_add() adiciona dados em um blob criado com ibase_blob_create().

Parâmetros

blob_handle

Um manipulador blob aberto com ibase_blob_create().

data

A informação a ser adicionada.

Valor Retornado

Não há valor retornado.

Veja Também

  • ibase_blob_cancel() - Cancela a criação de um blob
  • ibase_blob_close() - Fecha o blob
  • ibase_blob_create() - Cria um novo blob para adicionar dados
  • ibase_blob_import() - Cria um blob, copia um arquivo para ele e o fecha
