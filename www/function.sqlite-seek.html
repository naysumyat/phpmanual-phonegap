sqlite_seek

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_seek — Move o ponteiro interno para uma linha em um conjunto de resultados

Descrição

bool sqlite_seek ( resource $result , int $rownum )

sqlite_seek() move o ponteiro interno para a linha idicada pelo parâmetro rownum. O numero da linha é baseado em zero (0 é a primeira linha). Retorna FALSE se a linha não existir, TRUE em outro caso.

Nota:

Esta função não pode ser usada com query sem buffer.

Veja também sqlite_next(), sqlite_current() e sqlite_rewind().
