mt_getrandmax

(PHP 4, PHP 5)

mt_getrandmax — Retorna o maior valor aletório possível

Descrição

int mt_getrandmax ( void )

Retorna o maior valor que pode ser obtido numa chamada a mt_rand().

Valor Retornado

Retorna o valor máximo aleatório retornado por mt_rand()
