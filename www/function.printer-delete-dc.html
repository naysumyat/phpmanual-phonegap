printer_delete_dc

(PECL printer SVN)

printer_delete_dc — Exclui um device context

Descrição

bool printer_delete_dc ( resource $handle )

A função exclui um device context. Retorna TRUE em caso de sucesso ou FALSE em caso de falha. Para um exemplo veja printer_create_dc(). handle deve ser um manipulador válido para uma impressora.
