fam_next_event

(PHP 5 <= 5.0.5)

fam_next_event — Pega o próximo evento FAM pendente

Descrição

array fam_next_event ( resource $fam )

Retorna o próximo evento FAM pendente.

A função irá bloquear até que um evento que está disponível possa ser checado usando fam_pending().

Parâmetros

fam

Um resource representando a conexão para o serviço FAM retornado pela fam_open()

Valor Retornado

Retorna um array que contém um código de evento FAM no elemento 'code', o caminho do arquivo deste evento ficam no elemento 'filename' e opcionalmente o hostname no elemento 'hostname'.

Os possíveis códigos do evento são descritos em detalhes na parte de constantes dessa seção.
