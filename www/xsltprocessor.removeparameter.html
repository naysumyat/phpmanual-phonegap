XSLTProcessor::removeParameter

(PHP 5)

XSLTProcessor::removeParameter — Remove parâmetro

Descrição

bool removeParameter ( string $namespaceURI , string $localName )

Remove um parâmetro, se definido. Isto irá fazer o processador usar o valor padrão para o parâmetro como especificado no stylesheet.

Parâmetros

namespaceURI

O namespace URI do parâmetro XSLT.

localName

O nome local do parâmetro XSLT.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
