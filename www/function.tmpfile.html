tmpfile

(PHP 4, PHP 5)

tmpfile — Cria um arquivo temporário

Descrição

resource tmpfile ( void )

Cria um arquivo temporário com um nome único em modo de leitura-gravação (w+) e retorna o manipulador do arquivo.

Para detalhes consulte a documentação do seu sistem na função tmpfile(3), assim como o header stdio.h.

Valor Retornado

Retorna o manipulador de arquivo, similar ao retornado pela fopen(), para o novo arquivo, ou FALSE em falha.

Exemplos

Exemplo #1 Exemplo da tmpfile()

<?php
$temp = tmpfile();
fwrite($temp, "escrevendo no arquivo temporario");
fseek($temp, 0);
echo fread($temp, 1024);
fclose($temp); // isto remove o arquivo
?>
