mysqli_connect_error

(PHP 5)

mysqli_connect_error — Retorna uma string descrevendo o ultimo erro da função connect

Descrição

string mysqli_connect_error ( void )

A função mysqli_connect_error() é identica a função correspondente mysqli_connect_errno() de todas as maneiras, exceto que ao invés de retornar um número inteiro como código de erro, a função mysqli_connect_error() irá retornar uma string representando o ultimo erro que aconteceu com a ultima chamada a função mysqli_connect(). Se não houve erro, esta função irá retornar uma string vazia.

Valores de retorno

Uma string que descreve o erro. Uma string vazia se não houve erro.

Veja também

mysqli_connect(), mysqli_connect_errno(), mysqli_errno(), mysqli_error(), mysqli_sqlstate()

Exemplo

Exemplo #1 mysqli_connect_error sample

<?php

$link = @mysqli_connect("localhost", "nonexisting_user", "");

if (!$link) {
    printf("Can't connect to localhost. Error: %s\n", mysqli_connect_error());
}
?>
