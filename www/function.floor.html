floor

(PHP 4, PHP 5)

floor — Arredonda frações para baixo

Descrição

float floor ( float $value )

Retorna o próximo menor valor inteiro ao se arredondar para baixo o valor, se necessário.

Parâmetros

number

O valor numérico arredondado

Valor Retornado

value arredondado para o inteiro menor. O valor retornado de floor() é do tipo float porque o tipo float normalmente abrange mais números possíveis do que integer.

Exemplos

Exemplo #1 Exemplos de floor()

<?php
echo floor(4.3);   // 4
echo floor(9.999); // 9
echo floor(-3.14); // -4
?>
