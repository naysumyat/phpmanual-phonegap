Como nós geramos os formatos

Este Manual é escrito em XML usando o» DocBook XML DTD, usando » PhD (The [PH]P based [D]ocBook renderer) para manutenção e formatação.

Usando XML como formato, nos da a capacidade para gerar diversos outros formatos de saída a partir de seu arquivo fonte, ao mesmo tempo que a manutenção é feita em apenas um único lugar para todos os formatos. A ferramente usada para formatar o manual online é » PhD. Usamos » Microsoft HTML Help Workshop para gerar o formato Windows HTML Help do manual e, é claro, o próprio PHP para fazermos algumas conversões e formatações.

O manual PHP é gerado em várias linguagens e formatos, veja » http://www.php.net/docs.php para maiores detalhes. A código fonte XML pode ser baixado da SVN e visualizado em » http://svn.php.net/viewvc/. A documentação é no módulo phpdoc.
