readfile

(PHP 4, PHP 5)

readfile — Lê e exibe o conteúdo de um arquivo

Descrição

int readfile ( string $nomedoarquivo [, bool $use_include_path [, resource $context ]] )

Lê um arquivo e escreve o seu conteúdo para o buffer de saída (output buffer).

Parâmetros

filename

O nome do arquivo a ser lido.

use_include_path

Você pode usar o segundo parâmetro opcional e o configurar para TRUE, se você quiser procurar o arquivo no include_path também.

context

A context stream resource.

Valor Retornado

Retorna o número de bytes lidos do arquivo. Se um erro ocorrer, FALSE é retornado a menos que a função seja chamada com @readfile(), um erro é imprimido.

Notas

Dica

Você pode usar uma URL como um nome de arquivo nesta função se fopen wrappers estiver habilitado. Veja fopen() para mais detalhes em como especificar o nome do arquivo e Supported Protocols and Wrappers para uma lista de protocolos URL suportados.

Nota: O suporte ao contexto foi adicionado no PHP 5.0.0. Para uma descrição de contextos, veja Streams.
