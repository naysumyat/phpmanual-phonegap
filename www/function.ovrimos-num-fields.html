ovrimos_num_fields

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_num_fields — Retorna o número de colunas

Descrição

int ovrimos_num_fields ( int $result_id )

Retorna o número de colunas no especificado identificador de resultado.

Parâmetros

result_id

Um identificador de resultado, retornado pela ovrimos_execute() ou ovrimos_exec().

Valor Retornado

Retorna o número de colunas como um inteiro, ou FALSE em erro.
