apache_get_modules

(PHP 4 >= 4.3.2, PHP 5)

apache_get_modules — Retorna uma lista de módulos do Apache carregados

Descrição

array apache_get_modules ( void )

Retorna uma lista de módulos do Apache carregados.

Valor Retornado

Um array de módulos do Apache carregados.

Changelog

Versão Descrição
5.0.0 Tornou-se disponível usando o Apache 1 ou a API filter do Apache 2 . Antes disso, só estava disponível usando a API handler do Apache2.

Exemplos

Exemplo #1 Exemplo de uso da função apache_get_modules()

<?php
print_r(apache_get_modules());
?>

O exemplo acima irá imprimir algo similar à:

Array
(
    [0] => core
    [1] => http_core
    [2] => mod_so
    [3] => sapi_apache2
    [4] => mod_mime
    [5] => mod_rewrite
)
