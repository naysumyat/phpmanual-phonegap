gethostbyaddr

(PHP 4, PHP 5)

gethostbyaddr — Obtém nome do host de Internet correspondendo ao endereçõ de IP fornecido.

Descrição

string gethostbyaddr ( string $ip_address )

Retorna o nome do host do host de Internet especificado por ip_address.

Parâmetros

ip_address

O endereço IP do host.

Valor Retornado

Retorna o nome do host ou o não modificado ip_address em falha.

Exemplos

Exemplo #1 Um simples exemplo de gethostbyaddr()

<?php
$hostname = gethostbyaddr($_SERVER['REMOTE_ADDR']);
  
echo $hostname;
?>
