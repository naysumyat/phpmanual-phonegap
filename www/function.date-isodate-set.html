date_isodate_set

(PHP 5 >= 5.2.0)

date_isodate_set — Define a data ISO

Descrição

void date_isodate_set ( DateTime $object , int $year , int $week [, int $day ] )
void DateTime::setISODate ( int $year , int $week [, int $day ] )

Parâmetros

object

Objeto DateTime.

year

Ano da data.

week

Semana da data.

day

Dia da data.

Valor Retornado

Retorna NULL em caso de sucesso ou FALSE em falha.
