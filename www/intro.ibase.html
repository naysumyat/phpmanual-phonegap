Introdução

Firebird/InterBase é um banco de dados relacional que oferece muitos recursos ANSI SQL-92 e é executado em Linux, Windows, e uma variedade de plataformas Unix. Firebird/InterBase oferece excelente concorrência, alta peformance, e suporte a poderosa linguagem para stored procedures e triggers. Ele vem sendo usado em sistemas de produção, sob uma variedades de nomes desde 1981.

InterBase é o nome da variante de código fechado deste RDBMS que foi desenvolvido pela Borland/Inprise. Mais informação sobre InterBase está disponível em » http://www.borland.com/interbase/.

Firebird é um projeto comercialmente independente de programadores C e C++, conselheiros técnicos e suportes de desenvolvimento e revisores um sistema gerenciador de banco de dados relacional baseado no código fonte liberado por Inprise Corp (agora conhecido como Borland Software Corp) sob InterBase Public License v.1.0 em 25 Julho, 2000. Mais informação sobre Firebird está disponível em » http://www.firebirdsql.org/.

Nota:

Esta extensão suporta as versões 5 e superiores do InterBase e todas versões do Firebird. Suporte para versão 5.x do InterBase será removido no PHP 6.

Este banco de dados usa o caractere de aspa simples (') para escape, um comportamento similar ao do banco de dados Sybase, adicione ao seu php.ini a seguinte diretiva:

magic_quotes_sybase = On
