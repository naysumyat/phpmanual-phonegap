rmdir

(PHP 4, PHP 5)

rmdir — Remove um diretório

Descrição

bool rmdir ( string $dirname [, resource $context ] )

Tenta remover o diretório com o nome de dirname. O diretório tem que estar vazio e as permissões relevantes autorizarem a esta operação.

Parâmetros

dirname

Caminho para o diretório.

context

Nota: O suporte ao contexto foi adicionado no PHP 5.0.0. Para uma descrição de contextos, veja Streams.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Changelog

Versão Descrição
5.0.0 No PHP 5.0.0 rmdir() pode também ser usada com alguns URL wrappers. Consulte Supported Protocols and Wrappers para uma listagem de wrappers suportados pela rmdir().

Notas

Nota: Quando o safe-mode está ativo, o PHP verifica se o diretório no qual o script está em operação tem o mesmo UID (proprietário) do script que está sendo executado.
