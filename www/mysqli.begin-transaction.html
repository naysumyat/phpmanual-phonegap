mysqli::begin_transaction

mysqli_begin_transaction

(PHP 5 >= 5.5.0)

mysqli::begin_transaction -- mysqli_begin_transaction — Starts a transaction

Descrição

Estilo orientado à objeto (method):

public bool mysqli::begin_transaction ([ int $flags [, string $name ]] )

Estilo procedural:

bool mysqli_begin_transaction ( mysqli $link [, int $flags [, string $name ]] )

Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Parâmetros

link

Procedural style only: A link identifier returned by mysqli_connect() or mysqli_init()

flags

name

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Veja Também

  • mysqli_autocommit() - Ativa ou desativa o salvar automaticamente as modificações no banco de dados
  • mysqli_commit() - Salva a transação atual
  • mysqli_rollback() - Rolls back current transaction
