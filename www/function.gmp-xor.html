gmp_xor

(PHP 4 >= 4.0.4, PHP 5)

gmp_xor — Operação binária XOR

Descrição

resource gmp_xor ( resource $a , resource $b )

Faz a operação binária exclusiva OR (XOR) de dois números GMP.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

b

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_xor()

<?php
$xor1 = gmp_init("1101101110011101", 2);
$xor2 = gmp_init("0110011001011001", 2);
       
$xor3 = gmp_xor($xor1, $xor2);
       
echo gmp_strval($xor3, 2) . "\n";
?>

O exemplo acima irá imprimir:

1011110111000100
