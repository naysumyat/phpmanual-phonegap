odbc_foreignkeys

(PHP 4, PHP 5)

odbc_foreignkeys — Retorna uma lista de chaves estrangeiras na tabela especificada ou uma lista de chaves estrangeiras em outras tabelas que se referem a chave primária na tabela especificada

Descrição

resource odbc_foreignkeys ( resource $connection_id , string $pk_qualifier , string $pk_owner , string $pk_table , string $fk_qualifier , string $fk_owner , string $fk_table )

odbc_foreignkeys() obtém informações sobre as chaves estrangeiras. Retorna um conjunto de resultados ODBC FALSE em caso de falha.

O conjunto de resultados tem as seguintes colunas:

  • PKTABLE_QUALIFIER
  • PKTABLE_OWNER
  • PKTABLE_NAME
  • PKCOLUMN_NAME
  • FKTABLE_QUALIFIER
  • FKTABLE_OWNER
  • FKTABLE_NAME
  • FKCOLUMN_NAME
  • KEY_SEQ
  • UPDATE_RULE
  • DELETE_RULE
  • FK_NAME
  • PK_NAME

Se pk_table contém um nome de tabela, odbc_foreignkeys() retorna um conjunto de resultados contendo a chave primária da tabela especificada e todas as chaves estrangeiras que se referem a ela.

Se fk_table contém um nome de tabela, odbc_foreignkeys() retorna um conjunto de resultados de todas as chaves estrangeiras da tabela especificada e as chaves primárias (em outras tabelas) as quais elas se referem.

Se ambos pk_table e fk_table contiverem nomes de tabelas, odbc_foreignkeys() retorna as chaves estrangeiras na tabela especificada em fk_table que se referam a chave primária na tabela especificada em pk_table. Isto deve ser uma chave primária no máximo.
