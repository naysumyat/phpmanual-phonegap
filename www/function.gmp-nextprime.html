gmp_nextprime

(PHP 5 >= 5.2.0)

gmp_nextprime — Busca o próximo número primo

Descrição

resource gmp_nextprime ( int $a )

Busca o próximo número primo

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Retorna o próximo número primo maior que a, como um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_nextprime()

<?php
$prime1 = gmp_nextprime(10); // next prime number greater than 10
$prime2 = gmp_nextprime(-1000); // next prime number greater than -1000

echo gmp_strval($prime1) . "\n";
echo gmp_strval($prime2) . "\n";
?>

O exemplo acima irá imprimir:

11
-997

Notas

Nota:

Esta função usa um algoritmo probabilístico para identificar primos e chances ter um número composto é extremamente pequena.
