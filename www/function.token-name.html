token_name

(PHP 4 >= 4.2.0, PHP 5)

token_name — Pega o nome simbólico de um dado token do PHP

Descrição

string token_name ( int $token )

token_name() Pega o nome simbólico para o token do PHP.

Parâmetros

token

O valor do token.

Valor Retornado

O nome simbólico do token passado como parâmetro. O nome retornado é o mesmo nome da constante definida pelo token.

Exemplos

Exemplo #1 Exemplo de token_name()

<?php
// 260 é o valor do token T_REQUIRE 
echo token_name(260);        // -> "T_REQUIRE"

// uma constante token é igual ao seu próprio nome
echo token_name(T_FUNCTION); // -> "T_FUNCTION"
?>
