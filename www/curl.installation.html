Instalação

Para usar o suporte a cURL no PHP, é necessário compilar o PHP com a opção --with-curl[=DIR] onde DIR é o diretório que contém os diretórios lib e include. No diretório include deve haver uma pasta chamada curl que deve conter os arquivos easy.h e curl.h. Deve haver um arquivo chamado libcurl.a dentro do diretório lib. A partir do PHP 4.3.0 você pode configurar o PHP para utilizar cURL para streams com a opção --with-curlwrappers .

Nota: Observação para os usuários de Win32
Para ativar este módulo em ambiente Windows, libeay32.dll e ssleay32.dll precisam estar presentes no seu PATH. Você não precisa da libcurl.dll do site do cURL.
