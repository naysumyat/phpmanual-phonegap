lstat

(PHP 4, PHP 5)

lstat — Obtém informações sobre um arquivo ou link simbólico

Descrição

array lstat ( string $filename )

Pega as estatísticas do arquivo ou link simbólico com o nome de filename.

Parâmetros

filename

Path to a file or a symbolic link.

Valor Retornado

Veja a página do manual para stat() para informações sobre a estrutura do array retornado que lstat() retorna. Esta função é identica a stat() exceto que se o parâmetro filename é um link simbólico, e o status do link simbólico é retornado e não o status do arquivo apontando pelo link simbólico.

Notas

Nota: Os resultados desta função são cacheados. Veja clearstatcache() para mais detalhes.

Dica

A partir do PHP 5.0.0, esta função também pode ser utilizada com alguns wrappers URL. Veja Supported Protocols and Wrappers para uma lista de quais wrappers são suportados pela família de funções stat().
