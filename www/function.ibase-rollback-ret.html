ibase_rollback_ret

(PHP 5)

ibase_rollback_ret — Desfaz uma transação sem fechá-la

Descrição

bool ibase_rollback_ret ([ resource $link_or_trans_identifier ] )

Desfaz a transação sem fechá-la.

Parâmetros

link_or_trans_identifier

Se chamado sem um argumento, esta função desfaz a transação padrão do link padrão. Se o argumento é um identificador de conexão, a transação padrão da conexão correspondente será desfeita. Se o argumento é um identificador de transação, a transação correspondente será desfeita. O contexto da transação será retido, então comando executados de dentro desta transação não serão invalidados.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
