Introdução

A extensão Solr, permite comunicar-se de forma eficaz com um servidor Apache Solr ultilizando o PHP 5.

A extensão Solr é extremamente rápida, leve, rica em recursos de biblioteca que permite que os desenvolvedores de PHP se comuniquem de forma eficaz com servidores Solr.

Ele é compatível com as versões 1.3 e 1.4 do Apache Solr.

Ele tem ferramentas para adicionar documentos e fazer atualizações para o servidor Solr.

Ele também contém ferramentas que lhe permite construir consultas avançadas para o servidor para buscar documentos.
