ovrimos_execute

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_execute — Executa um comando SQL preparado

Descrição

bool ovrimos_execute ( int $result_id [, array $parameters_array ] )

ovrimos_execute() executa um comando preparado. Retorna TRUE em caso de sucesso ou FALSE em caso de falha. Se o comando preparado conter parâmetros (pontos de interogação no comando), o número correto de parâmetros deve ser passado em uma matriz. Veja que eu não acompanho a convenção do PHP em colocar o nome do parâmetro opcional entre conchetes. Eu não gosto disso.
