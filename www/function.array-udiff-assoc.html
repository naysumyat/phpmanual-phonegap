array_udiff_assoc

(PHP 5)

array_udiff_assoc — Computa a diferença entre arrays com checagem adicional de índice, compara dados por uma função de callback

Descrição

array array_udiff_assoc ( array $array1 , array $array2 [, array $ ... ], callback $data_compare_func )

Computa a diferença entre arrays com checagem adicional de índice, compara dados por uma função de callback.

Nota: Por favor note que esta função somente checa uma dimensão de um array n-dimensional. Obviamente você pode checar dimensões mais profundas usando, por exemplo, array_udiff_assoc($array1[0], $array2[0], "some_comparison_func"); .

Parâmetros

array1

O primeiro array.

array2

O segundo array.

data_compare_func

A função de callback de comparação.

A função de callback fornecida pelo usuário é usada na comparação. Ela deve retornar um inteiro menor que, igual a, ou maior que zero se o primeiro argumento é considerado respectivamente menor que, igual a, ou maior que o segundo.

Valor Retornado

array_udiff_assoc() retorna um array contendo todos os valores de array1 que não estão presentes em qualquer dos outros argumentos. Note que as chaves são usadas na comparação, ao contrário de array_diff() e array_udiff(). A comparação dos dados dos arrays é executada usando uma função de callback fornecida pelo usuário. Neste aspecto o comportamente é oposto ao de array_diff_assoc() que usa função interna para comparação.

Exemplos

Exemplo #1 Exemplo de array_udiff_assoc()

<?php
class cr {
    private $priv_member;
    function cr($val)
    {
        $this->priv_member = $val;
    }

    static function comp_func_cr($a, $b)
    {
        if ($a->priv_member === $b->priv_member) return 0;
        return ($a->priv_member > $b->priv_member)? 1:-1;
    }
}

$a = array("0.1" => new cr(9), "0.5" => new cr(12), 0 => new cr(23), 1=> new cr(4), 2 => new cr(-15),);
$b = array("0.2" => new cr(9), "0.5" => new cr(22), 0 => new cr(3), 1=> new cr(4), 2 => new cr(-15),);

$result = array_udiff_assoc($a, $b, array("cr", "comp_func_cr"));
print_r($result);
?>

O exemplo acima irá imprimir:

Array
(
    [0.1] => cr Object
        (
            [priv_member:private] => 9
        )

    [0.5] => cr Object
        (
            [priv_member:private] => 12
        )

    [0] => cr Object
        (
            [priv_member:private] => 23
        )
)

Em nosso exemplo acima você vê que o par "1" => new cr(4) está presente em ambos os arrays e assim ele não está presente na saída da função.
