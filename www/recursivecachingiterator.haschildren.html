RecursiveCachingIterator::hasChildren

(PHP 5 >= 5.1.0)

RecursiveCachingIterator::hasChildren — Verifica se o elemento atual do iterador interno possui filhos ou não

Descrição

bolean RecursiveCachingIterator::hasChildren ( void )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

TRUE se o iterador interno tem filhos, FALSE caso contrário
