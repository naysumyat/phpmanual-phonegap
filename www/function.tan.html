tan

(PHP 4, PHP 5)

tan — Tangente

Descrição

float tan ( float $arg )

tan() retorna a tangente do parâmetro arg. O argumento arg deve estar em radianos.

Parâmetros

arg

O argumento para processar em radianos

Valor Retornado

A tangente de arg

Exemplos

Exemplo #1 Exemplo da tan()

<?php

echo tan(M_PI_4); // 1

?>
