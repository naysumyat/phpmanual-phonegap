basename

(PHP 4, PHP 5)

basename — Retorna a parte nome do arquivo do caminho/path

Descrição

string basename ( string $path [, string $suffix ] )

Dado uma string contendo um caminho para um arquivo, essa função irá retornar o nome base do arquivo.

Parâmetros

path

A path.

No Windows, tanto a barra (/) quanto a barra invertida (\) são usadas como caracter de separacao do caminho. Em outros ambientes, somente a barra (/).

suffix

Se o nome do arquivo termina em suffix ele será removido.

Valor Retornado

Retorna o nome base de um dado path.

Changelog

Versão Descrição
4.1.0 O parâmetro suffix foi adicionado

Exemplos

Exemplo #1 Exemplo da basename()

<?php
$caminho = "/home/httpd/html/index.php";
$arquivo = basename ($caminho);        // $arquivo = "index.php"
$arquivo = basename ($caminho,".php"); // $arquivo = "index"
?>
