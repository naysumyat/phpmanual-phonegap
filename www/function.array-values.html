array_values

(PHP 4, PHP 5)

array_values — Retorna todos os valores de um array

Descrição

array array_values ( array $input )

array_values() retorna todos os valores do array input num array indexado numericamente.

Parâmetros

input

O array.

Valor Retornado

Retorna um array indexado de valores.

Exemplos

Exemplo #1 Exemplo de array_values()

<?php
$array = array("tamanho" => "G", "cor" => "dourado");
print_r(array_values ($array));
?>

O exemplo acima irá imprimir:

Array
(
    [0] => G
    [1] => dourado
)
