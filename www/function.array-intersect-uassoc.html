array_intersect_uassoc

(PHP 5)

array_intersect_uassoc — Computa a interseção de arrays com checagem de índice adicional, compara índices por uma função de callback

Descrição

array array_intersect_uassoc ( array $array1 , array $array2 [, array $ ... ], callback $key_compare_func )

array_intersect_uassoc() retorna um array contendo todos os valores de array1 que estão presentes em todos os argumentos. Note que as chaves são usadas na comparação, ao contrário de array_intersect().

A comparação de índices é feita por uma função de callback fornecida pelo usuário. Ela deve retornar um inteiro menor que, igual a, ou maior que zero se o primeiro argumento é considerado respectivamente, menor que, igual a, ou maior que o segundo.

Parâmetros

array1

Array inicial para comparação dos arrays.

array2

Primeiro array a ser comparado.

array

Lista variável de argumentos de arrays para comparar valores.

key_compare_func

Função de callback fornecida pelo usuário para fazer a comparação.

Valor Retornado

Retorna os valores de array1 cujos valores existem em todos os argumentos.

Exemplos

Exemplo #1 Exemplo de array_intersect_uassoc()

<?php
$array1 = array("a" => "verde", "b" => "marrom", "c" => "azul", "vermelho");
$array2 = array("a" => "VERDE", "B" => "marrom", "amarelo", "vermelho");

print_r(array_intersect_uassoc($array1, $array2, "strcasecmp"));
?>

O exemplo acima irá imprimir:

Array
(
    [b] => marrom
)
