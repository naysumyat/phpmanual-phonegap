is_bool

(PHP 4, PHP 5)

is_bool — Verifica se a variável é um boleano

Descrição

bool is_bool ( mixed $var )

Verifica se uma dada variável é boolean.

Parâmetros

var

The variable being evaluated.

Valor Retornado

Retorna TRUE se var é um boolean, FALSE caso contrário.

Exemplos

Exemplo #1 Exemplo is_bool()

<?php
$a = false;
$b = 0;

// Verifica se $a é boolean, retorna true
if (is_bool($a)) {
    echo 'A variável $a é um valor booleano';
}

// Verifica se $b é boolean, retorna false
if (is_bool($b)) {
    echo 'A variável $b é um valor booleano';
}
?>
