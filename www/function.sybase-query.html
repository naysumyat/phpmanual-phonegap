sybase_query

(PHP 4, PHP 5)

sybase_query — Envia uma consulta Sybase

Descrição

resource sybase_query ( string $query , resource $link_identifier )

Retorna um identificador de reultado Sybase positivo em caso de sucesso, ou FALSE em caso de erro.

sybase_query() envia uma consulta para o banco de dados ativo no servidor que esta associado ao identificador de conexão especificado. Se o identificador de conexão não for especificado, a última conexão aberta é usada. Se não houver conexão aberta, a função tenta estabelecer uma como se sybase_connect() fosse utilizada, e a utiliza.

Veja também sybase_select_db() e sybase_connect().
