Introdução

Em adição ao suporte ODBC normal, as funções de ODBC Unificadas no PHP permitem que você acesse vários bancos de dados que emprestaram a semântica da API do ODBC para implementer a sua própria API. Ao invés de manter múltiplos drivers de banco de dados que são todos praticamente identicos, estes drivers foram unificados em um único conjunto de funções ODBC.

Os seguintes bancos de dados são suportados pelas funções de OBDC Unificado: » Adabas D, » IBM DB2, » iODBC, » Solid, e » Sybase SQL Anywhere.

Nota:

Com a excessão de iODBC, não existe ODBC involvido ao conectar aos bancos de dados acima. As funções que você usa para conversar nativamente com eles apenas compartilham os mesmos nomes e sintaxe das funções ODBC. Entretanto, compilar o PHP com suporte a iODBC permite a você usar qualquer dirver compatível com ODBC nas suas aplicações PHP. Maiores informações sobre iODBC, estão disponíveis em » www.iodbc.org com a alternativa unixODBC disponível em » www.unixodbc.org.
