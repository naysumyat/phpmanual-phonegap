fam_monitor_directory

(PHP 5 <= 5.0.5)

fam_monitor_directory — Monitora alterações em diretórios

Descrição

resource fam_monitor_directory ( resource $fam , string $dirname )

Requisita monitoramento para um diretório e todos os arquivos contidos nele.

Um evento FAM irá ser gerado quando o status do diretório (ex.: o resultado da função stat() nesse diretório) ou isso (ex. o resultado de readdir()) acontecerem.

Os possíveis códigos do evento são descritos em detalhes na parte de constantes dessa seção.

Parâmetros

fam

Um resource representando a conexão para o serviço FAM retornado pela fam_open()

dirname

Caminho para o diretório a ser monitorado

Valor Retornado

Retorna um resource de monitoração ou FALSE em erros.
