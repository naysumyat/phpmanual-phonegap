filepro_fieldtype

(PHP 4, PHP 5 <= 5.1.6)

filepro_fieldtype — Obtém o tipo de um campo

Descrição

string filepro_fieldtype ( int $field_number )

Retorna o tipo de campo correspondente a field_number.

Parâmetros

field_number

O número do campo.

Valor Retornado

Retorna o tipo de edição do campo como uma string, ou FALSE em caso de erro.
