printer_delete_font

(PECL printer SVN)

printer_delete_font — Exclui uma fonte

Descrição

bool printer_delete_font ( resource $handle )

Esta função exclui a fonte selecionada. Para um exemplo veja printer_select_font(). Retorna TRUE em caso de sucesso ou FALSE em caso de falha. handle deve ser um manipulador válido para a fonte.
