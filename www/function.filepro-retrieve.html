filepro_retrieve

(PHP 4, PHP 5 <= 5.1.6)

filepro_retrieve — Obtém dados de um banco de dados filePro

Descrição

string filepro_retrieve ( int $row_number , int $field_number )

Retorna dados do local especificado no banco de dados.

Nota: Quando o safe-mode está ativo, o PHP verifica se os arquivo(s) ou diretórios que estão usados na operação tem o mesmo UID (proprietário) do script que está sendo executado.

Parâmetros

row_number

O número da linha. Deve ser entre zero e o número total de linhas menos um (0..filepro_rowcount() - 1)

field_number

O número do campo. Aceita valores entre zero e o número total de campos menos um (0..filepro_fieldcount() - 1)

Valor Retornado

Retorna os dados especificados ou FALSE em caso de erro.
