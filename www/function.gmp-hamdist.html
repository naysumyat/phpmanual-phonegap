gmp_hamdist

(PHP 4 >= 4.0.4, PHP 5)

gmp_hamdist — Distância de Hamming

Descrição

int gmp_hamdist ( resource $a , resource $b )

Retorna a distância de Hamming entre a e b. Ambos operandos devem ser positivo.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Tem que ser positivo.

b

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Tem que ser positivo.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_hamdist()

<?php
$ham1 = gmp_init("1001010011", 2);
$ham2 = gmp_init("1011111100", 2);
echo gmp_hamdist($ham1, $ham2) . "\n";

/* hamdist is equivilent to: */
echo gmp_popcount(gmp_xor($ham1, $ham2)) . "\n";
?>

O exemplo acima irá imprimir:

6
