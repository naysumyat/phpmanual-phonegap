imagesetpixel

(PHP 4, PHP 5)

imagesetpixel — Define um único pixel

Descrição

int imagesetpixel ( resource $image , int $x , int $y , int $color )

imagesetpixel() desenha um pixel em x, y (em cima na esquerda é 0, 0) na imagem image com a cor color.

Veja também imagecreate() e imagecolorallocate().
