ngettext

(PHP 4 >= 4.2.0, PHP 5)

ngettext — O mesmo que gettext, mas você pode especificar mais de uma mensagem

Descrição

string ngettext ( string $msgid1 , string $msgid2 , int $n )

A versão de plural da gettext(). Algumas linguagens tem mais de uma forma para mensagens no plural dependendo da quantidade.

Parâmetros

msgid1

msgid2

n

Valor Retornado

Retorna o plural na forma correta para uma identificada mensagem por msgid1 e msgid2 para quantidade n.

Exemplos

Exemplo #1 Exemplo da ngettext()

<?php

setlocale(LC_ALL, 'cs_CZ');
printf(ngettext("%d window", "%d windows", 1), 1); // 1 okno
printf(ngettext("%d window", "%d windows", 2), 2); // 2 okna
printf(ngettext("%d window", "%d windows", 5), 5); // 5 oken

?>
