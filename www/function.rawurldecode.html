rawurldecode

(PHP 4, PHP 5)

rawurldecode — Decodifica uma URL

Descrição

string rawurldecode ( string $str )

Retorna uma string em que as sequencias com o sinal porcento (%) seguida por dois digitos hexadecimais são substituidas pelos caracteres correspondentes.

Parâmetros

str

A URL a ser decodificada.

Valor Retornado

Retorna a URL decodificada, como uma string.

Exemplos

Exemplo #1 Exemplo da rawurldecode()

<?php

echo rawurldecode('foo%20bar%40baz'); // foo bar@baz

?>

Notas

Nota:

rawurldecode() não decodifica os sinais de adição ('+') em espaços. A função urldecode() faz isto.
