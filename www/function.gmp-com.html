gmp_com

(PHP 4 >= 4.0.4, PHP 5)

gmp_com — Calcula o complemento

Descrição

resource gmp_com ( resource $a )

Retorna o complemento de a

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Retorna o complemento de a, como um número GMP.

Exemplos

Exemplo #1 Exemplo gmp_com()

<?php
$com = gmp_com("1234");
echo gmp_strval($com) . "\n";
?>

O exemplo acima irá imprimir:

-1235
