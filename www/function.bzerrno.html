bzerrno

(PHP 4 >= 4.0.4, PHP 5)

bzerrno — Retorna um número de erro do bzip2

Descrição

int bzerrno ( resource $bz )

Retorna o número do erro de qualquer erro do bzip2 retornado pelo ponteiro de arquivo bz.

Veja também bzerror() e bzerrstr().
