lzf_decompress

(PECL lzf >= 0.1.0)

lzf_decompress — Descompressão LZF

Descrição

string lzf_decompress ( string $data )

lzf_compress() decomprime a string data contendo dados com codificação lzf.

Parâmetros

data

A string comprimida.

Valor Retornado

Retorna a informação decompressa ou FALSE se um erro ocorreu.
