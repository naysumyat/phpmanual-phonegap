spl_autoload

(PHP 5 >= 5.1.2)

spl_autoload — Implementação padrão de __autoload()

Descrição

void spl_autoload ( string $class_name [, string $file_extensions ] )

Esta funçao é para ser usada como implementação padrão para __autoload(). Se nenhuma outra função for especificada e a autoload_register() for chamada sem argumentos, esta função será usada para qualquer posterior chamada ao __autoload(). Por padrão, ela verifica todos os caminhos de inclusão por arquivos formados pelo nome da classe em minúsculo acrescido das extensões de arquivo .inc e .php.
