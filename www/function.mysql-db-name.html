mysql_db_name

(PHP 4, PHP 5)

mysql_db_name — Obtém dados do resultado(nome de banco de dados)

Descrição

string mysql_db_name ( resource $result , int $row [, mixed $field ] )

Obtém o nome do banco de dados a partir de uma chamada a mysql_list_dbs().

Parâmetros

result

O ponteiro de resultado de uma chamada a mysql_list_dbs().

row

O índice dentro do conjunto de resultados.

field

O nome do campo.

Valor Retornado

Retorna o nome do banco de dados em caso de sucesso, e FALSE em caso de falha. Se for retornado FALSE, use mysql_error() para determinar a natureza do erro.

Exemplos

Exemplo #1 Exemplo mysql_db_name()

<?php
error_reporting(E_ALL);

$link = mysql_connect('dbhost', 'username', 'password');
$db_list = mysql_list_dbs($link);

$i = 0;
$cnt = mysql_num_rows($db_list);
while ($i < $cnt) {
    echo mysql_db_name($db_list, $i) . "\n";
    $i++;
}
?>

Notas

Nota:

Para compatibilidade com versões anteriores, o seguinte apelido obsoleto pode ser usado: mysql_dbname()
