curl_errno

(PHP 4 >= 4.0.3, PHP 5)

curl_errno — Retorna o número do último erro

Descrição

int curl_errno ( resource $ch )

Retorna o número do erro para a última operação cURL.

Parâmetros

ch

Um manipulador cURL retornado por curl_init().

Valor Retornado

Retorna o número do erro ou 0 (zero) caso nenhum erro tenha acontecido.

Exemplos

Exemplo #1 Exemplo de curl_errno()

<?php
// Cria uma nova sessão para uma página não existente
$ch = curl_init('http://404.php.net/');

// Executa
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_exec($ch);

// Verifica se algum erro ocorreu
if(curl_errno($ch))
{
    echo 'Curl error: ' . curl_error($ch);
}

// Fecha a sessão
curl_close($ch);
?>
