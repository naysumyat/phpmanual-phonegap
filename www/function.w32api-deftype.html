w32api_deftype

(PHP 4 >= 4.2.0)

w32api_deftype — Define um tipo para ser usado com outras funções da w32api_functions

Descrição

bool w32api_deftype ( string $typename , string $member1_type , string $member1_name [, string $... [, string $... ]] )

Se você quiser definir um tipo para uma chamada à API Win32, você precisa chamar essa função.

Parâmetros

typename

The name of the type.

member1_type

Um tipo membro pode ser um tipo definido pelo usuário. Nomes de tipos nativos devem ser fornecidos em letras minúsculas.

member1_name

O nome do membro de member1_type.

...

...

Essa função recebe 2n+1 argumentos, onde n é o número de membros que o tipo tem. Depois disso é o tipo do membro seguido pelos nome dos membos (em pares).

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Notas

Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome, incluindo toda documentação pode ser modificado sem aviso em futuras versões do PHP. Esta função deve ser usada por sua própria conta e risco.
