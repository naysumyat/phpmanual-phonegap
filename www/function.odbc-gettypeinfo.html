odbc_gettypeinfo

(PHP 4, PHP 5)

odbc_gettypeinfo — Retorna um identificador de resultados contendo informações sobre os tipos de dados suportados pela fonte de dados

Descrição

resource odbc_gettypeinfo ( resource $connection_id [, int $data_type ] )

Obtém informações sobre os tipos de dados suportados pela fonte de dados. Retorna um identificador de resultados ODBC ou FALSE em caso de falha. O argumento opcional data_type pode ser utilizado para restringir a informação para um único tipo de dados.

O conjunto de resultados tem as seguintes colunas:

  • TYPE_NAME
  • DATA_TYPE
  • PRECISION
  • LITERAL_PREFIX
  • LITERAL_SUFFIX
  • CREATE_PARAMS
  • NULLABLE
  • CASE_SENSITIVE
  • SEARCHABLE
  • UNSIGNED_ATTRIBUTE
  • MONEY
  • AUTO_INCREMENT
  • LOCAL_TYPE_NAME
  • MINIMUM_SCALE
  • MAXIMUM_SCALE

O conjunto de resultados é ordenado por DATA_TYPE e TYPE_NAME.
