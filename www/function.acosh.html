acosh

(PHP 4 >= 4.1.0, PHP 5)

acosh — Cosseno Hiperbólico Inverso

Descrição

float acosh ( float $arg )

Retorna o cosseno hiperbólico de arg, isto é, o valor cujo coseno hiperbólico é arg.

Nota: esta função não é implementada na plataforma Windows

Parâmetros

arg

O valor para processar

Valor Retornado

O cosseno hiperbólico inverso do arg
