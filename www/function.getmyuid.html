getmyuid

(PHP 4, PHP 5)

getmyuid — Obtém o UID do dono do script PHP

Descrição

int getmyuid ( void )

Obtém o ID do usuário do atual script.

Valor Retornado

Retorna o ID do usuário do atual script, ou FALSE em erro.

  • getmyinode() - Obtém o inode do script atual
