md5

(PHP 4, PHP 5)

md5 — Calcula o "hash MD5" de uma string

Descrição

string md5 ( string $str [, bool $raw_output ] )

Calcula o "hash MD5" de str usando » RSA Data Security, Inc. MD5 Message-Digest Algorithm, e devolve esse hash.

Parâmetros

str

A string.

raw_output

Se o opcional raw_output é definido para TRUE, então o md5 é ao invés retornado no formato binário com um tamanho de 16. Padrão para FALSE.

Valor Retornado

Retorna o hash como um número hexadecimal de 32 caracteres.

Changelog

Versão Descrição
5.0.0 O parâmetro raw_output foi adicionado.

Exemplos

Exemplo #1 Um exemplo da md5()

<?php
$str = 'apple';

if (md5($str) === '1f3870be274f6c49b3e31a0c6728957f') {
    echo "Would you like a green or red apple?";
    exit;
}
?>
