ibase_modify_user

(PHP 5)

ibase_modify_user — Modifica um usuário para um banco de dados seguro (somente para IB6 ou posterior)

Descrição

bool ibase_modify_user ( resource $service_handle , string $user_name , string $password [, string $first_name [, string $middle_name [, string $last_name ]]] )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

O PHP 4 usa server, dba_user_name e dba_user_password ao invés do parâmetro service_handle.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Veja Também

  • ibase_add_user() - Adiciona um usuário a um banco de dados seguro (apenas para IB6 ou posterior)
  • ibase_delete_user() - Exclui um usuário de um banco de dados seguro (apenas para o IB6 ou posterior)
